# Démarrage rapide

Ce guide vous accompagne étape par étape pour configurer FluenzR et lancer votre première campagne d'emailing.

## Étape 1 : Créer votre compte

1. Rendez-vous sur [app.fluenzr.co](https://app.fluenzr.co)
2. Cliquez sur **S'inscrire**
3. Remplissez le formulaire avec votre email et mot de passe
4. Validez votre email (vérifiez vos spams si nécessaire)

## Étape 2 : Connecter votre compte email

FluenzR a besoin d'un compte email pour envoyer vos campagnes.

### Option A : Gmail (recommandé)

1. Dans le menu, cliquez sur **Comptes Email**
2. Cliquez sur **+ Gmail**
3. Connectez-vous à votre compte Google
4. Autorisez FluenzR à accéder à votre compte

> **ℹ️ Info**
Vous verrez un avertissement "Application non vérifiée". C'est normal, notre certification Google est en cours. Cliquez sur "Paramètres avancés" puis "Accéder à FluenzR".

### Option B : Outlook

1. Cliquez sur **+ Microsoft**
2. Connectez-vous à votre compte Microsoft
3. Autorisez les permissions demandées

### Option C : SMTP

Pour les autres fournisseurs, consultez notre guide [Configuration SMTP](/docs/email-accounts/smtp).

## Étape 3 : Ajouter vos contacts

### Création manuelle

1. Allez dans **Contacts**
2. Cliquez sur **+ Ajouter un contact**
3. Remplissez au minimum l'email
4. Cliquez sur **Enregistrer**

### Import CSV

Pour importer plusieurs contacts :

1. Cliquez sur **Importer CSV**
2. Sélectionnez votre fichier (format : email, firstName, lastName, company, ...)
3. Vérifiez l'aperçu
4. Confirmez l'import

> **⚠️ Attention**
Assurez-vous d'avoir le consentement de vos contacts avant de leur envoyer des emails commerciaux.

## Étape 4 : Créer une campagne

1. Allez dans **Campagnes**
2. Cliquez sur **+ Nouvelle campagne**
3. Donnez un nom à votre campagne
4. Configurez le workflow :

### Ajouter un email

1. Cliquez sur le nœud **Email** dans la barre d'outils
2. Rédigez votre objet et contenu
3. Utilisez des variables pour personnaliser : `{{firstName}}`, `{{company}}`

### Ajouter un délai

Pour créer une séquence multi-emails :

1. Ajoutez un nœud **Délai** (ex: 3 jours)
2. Ajoutez un second email
3. Répétez selon vos besoins

## Étape 5 : Ajouter des contacts à la campagne

1. Cliquez sur **Ajouter des contacts**
2. Sélectionnez un groupe ou des contacts individuels
3. Confirmez

## Étape 6 : Lancer la campagne

1. Vérifiez l'aperçu de votre email
2. Cliquez sur **Démarrer**
3. Confirmez le lancement

> **✅ Succès**
Félicitations ! Votre campagne est en cours. Suivez les statistiques en temps réel dans l'onglet **Statistiques**.
