# Importer des contacts

FluenzR vous permet d'importer des centaines ou milliers de contacts en quelques clics via un fichier CSV.

## Préparer votre fichier CSV

### Format requis

Votre fichier CSV doit respecter les règles suivantes :

- **Encodage** : UTF-8 (pour les accents)
- **Séparateur** : Virgule (,) ou point-virgule (;)
- **Première ligne** : En-têtes de colonnes

### Colonnes supportées

| Colonne CSV | Champ FluenzR | Obligatoire |
|-------------|---------------|-------------|
| `email` | Email | ✅ |
| `firstName` | Prénom | Non |
| `lastName` | Nom | Non |
| `company` | Entreprise | Non |
| `jobTitle` | Poste | Non |
| `phone` | Téléphone | Non |
| `website` | Site web | Non |
| `[champ_personnalisé]` | Champ personnalisé | Non |

### Exemple de fichier

```csv
email,firstName,lastName,company,jobTitle
jean.dupont@example.com,Jean,Dupont,Acme Corp,Directeur Commercial
marie.martin@startup.io,Marie,Martin,StartupIO,CEO
pierre.durand@tech.fr,Pierre,Durand,TechFrance,CTO
```

## Processus d'import

### 1. Accéder à l'import

1. Allez dans **Contacts**
2. Cliquez sur **Importer CSV**

### 2. Sélectionner le fichier

1. Cliquez sur **Choisir un fichier** ou glissez-déposez votre CSV
2. FluenzR analyse automatiquement le fichier

### 3. Mapper les colonnes

FluenzR détecte automatiquement les colonnes, mais vous pouvez ajuster :

1. Vérifiez que chaque colonne CSV correspond au bon champ FluenzR
2. Utilisez le menu déroulant pour corriger si nécessaire
3. Sélectionnez "Ignorer" pour les colonnes non pertinentes

### 4. Options d'import

| Option | Description |
|--------|-------------|
| **Groupe de destination** | Ajouter tous les contacts à un groupe |
| **Tags** | Appliquer des tags à tous les contacts |
| **Gestion des doublons** | Ignorer, mettre à jour, ou créer quand même |

### 5. Aperçu et validation

1. Vérifiez l'aperçu des premières lignes
2. FluenzR signale les erreurs potentielles (emails invalides, etc.)
3. Corrigez les problèmes ou acceptez d'ignorer les lignes en erreur

### 6. Lancer l'import

Cliquez sur **Importer** pour démarrer. Une barre de progression s'affiche.

## Gestion des doublons

Lorsqu'un email existe déjà, vous avez trois options :

| Option | Comportement |
|--------|--------------|
| **Ignorer** | Le contact existant n'est pas modifié |
| **Mettre à jour** | Les champs du contact sont mis à jour avec les nouvelles valeurs |
| **Créer quand même** | ⚠️ Crée un doublon (non recommandé) |

> **ℹ️ Info**
La détection de doublons se fait uniquement sur l'adresse email (insensible à la casse).

## Après l'import

### Rapport d'import

À la fin de l'import, un rapport affiche :

- Nombre de contacts importés
- Nombre de doublons détectés
- Nombre d'erreurs (avec détails)

### Vérification

1. Allez dans **Contacts**
2. Filtrez par le groupe ou tag utilisé lors de l'import
3. Vérifiez quelques contacts au hasard

## Limites et performances

| Plan | Limite par import |
|------|------------------|
| Free | 300 contacts |
| Starter | 10 000 contacts |
| Pro | 50 000 contacts |

> **⚠️ Attention**
Pour les imports volumineux (> 10 000 contacts), divisez votre fichier en plusieurs lots pour de meilleures performances.

## Résolution de problèmes

### "Encodage incorrect"

Symptôme : Caractères spéciaux illisibles (é devient Ã©)

Solution :
1. Ouvrez votre fichier dans Excel ou LibreOffice
2. Enregistrez sous > Format CSV UTF-8
3. Réimportez

### "Colonne email non détectée"

Solution :
1. Vérifiez que la première ligne contient les en-têtes
2. L'en-tête doit être exactement `email` (minuscules)
3. Mappez manuellement si nécessaire

### "Trop d'erreurs"

Si plus de 10% des lignes sont en erreur :
1. Téléchargez le rapport d'erreurs
2. Corrigez votre fichier source
3. Réimportez uniquement les lignes corrigées

### Import bloqué

Si l'import semble bloqué :
1. Attendez quelques minutes (les gros fichiers prennent du temps)
2. Actualisez la page
3. Vérifiez dans Contacts si des contacts ont été ajoutés
4. Contactez le support si le problème persiste

## Bonnes pratiques

### Avant l'import

- Nettoyez votre fichier (supprimez lignes vides, corrigez erreurs)
- Validez les emails avec un service externe si possible
- Préparez les groupes et tags à utiliser

### Pendant l'import

- Ne fermez pas la fenêtre pendant l'import
- Notez le nombre de contacts attendus pour vérifier

### Après l'import

- Vérifiez le rapport d'import
- Testez quelques contacts dans une campagne test
- Gardez une copie de votre fichier CSV original
