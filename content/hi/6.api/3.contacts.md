# संपर्क API

## संपर्क सूचीबद्ध करें

संपर्कों की पृष्ठांकित सूची प्राप्त करता है।

```
GET /contacts
```

### क्वेरी पैरामीटर

| पैरामीटर | प्रकार | विवरण |
|-----------|------|-------------|
| `page` | integer | पृष्ठ संख्या (डिफ़ॉल्ट: 1) |
| `perPage` | integer | प्रति पृष्ठ आइटम (डिफ़ॉल्ट: 20, अधिकतम: 100) |
| `email` | string | ईमेल द्वारा फ़िल्टर (शामिल) |
| `status` | string | स्थिति द्वारा फ़िल्टर |
| `groupId` | string | समूह द्वारा फ़िल्टर |
| `tag` | string | टैग द्वारा फ़िल्टर |
| `sort` | string | सॉर्ट फ़ील्ड |
| `order` | string | asc या desc |

### उदाहरण

```bash
curl -X GET "https://api.fluenzr.co/v1/contacts?groupId=grp_xyz789&status=active&perPage=50" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### प्रतिक्रिया

```json
{
  "success": true,
  "data": [
    {
      "id": "con_abc123",
      "email": "rahul.sharma@example.com",
      "firstName": "राहुल",
      ...
    },
    ...
  ],
  "meta": {
    "page": 1,
    "perPage": 50,
    "total": 234,
    "totalPages": 5
  }
}
```

## एक संपर्क प्राप्त करें

किसी विशिष्ट संपर्क का विवरण प्राप्त करता है।

```bash
  -H "Authorization: Bearer flz_sk_xxx"
```

### प्रतिक्रिया

```json
{
  "success": true,
  "data": {
    "id": "con_abc123",
    "email": "rahul.sharma@example.com",
    ...
  }
}
```

## संपर्क बनाएं

एक नया संपर्क बनाता है।

```
POST /contacts
```

### अनुरोध बॉडी

| फ़ील्ड | प्रकार | आवश्यक | विवरण |
|-------|------|----------|-------------|
| `email` | string | हां | ईमेल पता (अद्वितीय) |
| `firstName` | string | | प्रथम नाम |
| `lastName` | string | | अंतिम नाम |
| `company` | string | | कंपनी |
| `jobTitle` | string | | नौकरी का शीर्षक |
| `phone` | string | | फोन |
| `website` | string | | वेबसाइट |
| `groupIds` | array | | समूह IDs |
| `tags` | array | | टैग सूची |
| `customFields` | object | | कस्टम फ़ील्ड |
| `pipelineStageId` | string | | पाइपलाइन चरण ID |

### उदाहरण

```bash
curl -X POST "https://api.fluenzr.co/v1/contacts" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "priya.patel@startup.io",
    "firstName": "प्रिया",
    "lastName": "पटेल",
    "company": "StartupIO",
    "jobTitle": "CEO",
    "groupIds": ["grp_xyz789"],
    "tags": ["prospect", "saas"],
    "customFields": {
      "sector": "SaaS",
      "company_size": "10-50"
    }
  }'
```

### प्रतिक्रिया (201 Created)

```json
{
  "success": true,
  "data": {
    "id": "con_def456",
    "email": "priya.patel@startup.io",
    ...
  }
}
```

### संभावित त्रुटियां

| कोड | संदेश | विवरण |
|------|---------|-------------|
| 400 | Email is required | ईमेल गायब |
| 400 | Invalid email format | अमान्य ईमेल प्रारूप |
| 409 | Email already exists | ईमेल पहले से उपयोग में |

## संपर्क अपडेट करें

एक मौजूदा संपर्क को अपडेट करता है।

```
PUT /contacts/:id
```

### अनुरोध बॉडी

निर्माण के समान फ़ील्ड। केवल प्रदान किए गए फ़ील्ड अपडेट होते हैं।

### उदाहरण

```bash
curl -X PUT "https://api.fluenzr.co/v1/contacts/con_abc123" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "jobTitle": "VP Sales",
    "customFields": {
      "company_size": "100-500"
    }
  }'
```

### प्रतिक्रिया

```json
{
  "success": true,
  "data": {
    "id": "con_abc123",
    "jobTitle": "VP Sales",
    ...
  }
}
```

## संपर्क हटाएं

एक संपर्क को स्थायी रूप से हटाता है।

```
DELETE /contacts/:id
```

### उदाहरण

```bash
curl -X DELETE "https://api.fluenzr.co/v1/contacts/con_abc123" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### प्रतिक्रिया (200 OK)

```json
{
  "success": true,
  "data": {
    "id": "con_abc123",
    "deleted": true
  }
}
```

## बल्क निर्माण

एक ही अनुरोध में कई संपर्क बनाता है।

```
POST /contacts/bulk
```

### अनुरोध बॉडी

```json
{
  "contacts": [
    {
      "email": "contact1@example.com",
      "firstName": "Contact",
      "lastName": "One"
    },
    {
      "email": "contact2@example.com",
      "firstName": "Contact",
      "lastName": "Two"
    }
  ],
  "groupIds": ["grp_xyz789"],
  "onDuplicate": "skip"
}
```

| पैरामीटर | विवरण |
|-----------|-------------|
| `contacts` | संपर्कों की सरणी (अधिकतम 1000) |
| `groupIds` | सभी को असाइन करने के लिए समूह |
| `onDuplicate` | डुप्लिकेट पर कार्रवाई: `skip`, `update` |

### उदाहरण

```bash
curl -X POST "https://api.fluenzr.co/v1/contacts/bulk" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "contacts": [
      {"email": "a@example.com", "firstName": "A"},
      {"email": "b@example.com", "firstName": "B"}
    ],
    "onDuplicate": "skip"
  }'
```

### प्रतिक्रिया

```json
{
  "success": true,
  "data": {
    "created": 2,
    "updated": 0,
    "skipped": 0,
    "errors": []
  }
}
```

## संपर्क खोजें

उन्नत संपर्क खोज।

```
POST /contacts/search
```

### अनुरोध बॉडी

```json
{
  "query": "example.com",
  "filters": {
    "status": "active",
    "groups": ["grp_xyz789"],
    "customFields": {
      "sector": "Fintech"
    }
  },
  "sort": {
    "field": "createdAt",
    "order": "desc"
  }
}
```

### उदाहरण

```bash
curl -X POST "https://api.fluenzr.co/v1/contacts/search" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "filters": {
      "customFields": {
        "sector": "Fintech"
      }
    }
  }'
```

## संपर्क समूह प्रबंधित करें

### समूहों में जोड़ें

```
POST /contacts/:id/groups
```

```json
{
  "groupIds": ["grp_abc", "grp_def"]
}
```

### समूहों से हटाएं

```
DELETE /contacts/:id/groups
```

```json
{
  "groupIds": ["grp_abc"]
}
```
