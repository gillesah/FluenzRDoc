# REST API

FluenzR API आपको प्लेटफ़ॉर्म को अपने टूल के साथ एकीकृत करने और अपने वर्कफ़्लो को स्वचालित करने देता है।

> **जानकारी**
REST API केवल **Pro** योजना पर उपलब्ध है।

## अवलोकन

FluenzR API एक REST API है जो आपको अनुमति देता है:

- अपने संपर्कों को प्रोग्रामेटिक रूप से प्रबंधित करें
- अभियान बनाएं और नियंत्रित करें
- आंकड़े प्राप्त करें
- घटनाओं पर webhooks प्राप्त करें

## आधार URL

```
https://api.fluenzr.co/v1
```

## प्रतिक्रिया प्रारूप

सभी प्रतिक्रियाएं JSON प्रारूप में हैं:

```json
{
  "success": true,
  "data": { ... },
  "meta": {
    "page": 1,
    "perPage": 20,
    "total": 150
  }
}
```

### त्रुटि पर

```json
{
  "success": false,
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Email is required",
    "details": { ... }
  }
}
```

## HTTP कोड

| कोड | अर्थ |
|------|---------|
| 200 | सफलता |
| 201 | संसाधन बनाया गया |
| 400 | अमान्य अनुरोध |
| 401 | प्रमाणित नहीं |
| 403 | अधिकृत नहीं |
| 404 | संसाधन नहीं मिला |
| 429 | दर सीमा पार |
| 500 | सर्वर त्रुटि |

## दर सीमा

| योजना | सीमा |
|------|-------|
| Pro | 1000 अनुरोध/मिनट |

प्रतिक्रिया हेडर आपके उपयोग को इंगित करते हैं:

```
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 950
X-RateLimit-Reset: 1640000000
```

## पृष्ठांकन

संसाधन सूचीबद्ध करने वाले एंडपॉइंट पृष्ठांकन का समर्थन करते हैं:

```
GET /contacts?page=2&perPage=50
```

| पैरामीटर | विवरण | डिफ़ॉल्ट | अधिकतम |
|-----------|-------------|---------|-----|
| `page` | पृष्ठ संख्या | 1 | - |
| `perPage` | प्रति पृष्ठ आइटम | 20 | 100 |

## फ़िल्टरिंग

क्वेरी पैरामीटर के साथ परिणाम फ़िल्टर करें:

```
GET /contacts?email=example.com&group=123
```

## सॉर्टिंग

परिणाम क्रमबद्ध करें:

```
GET /contacts?sort=createdAt&order=desc
```

## उपलब्ध एंडपॉइंट

### संपर्क

| विधि | एंडपॉइंट | विवरण |
|--------|----------|-------------|
| GET | /contacts | संपर्क सूचीबद्ध करें |
| GET | /contacts/:id | संपर्क प्राप्त करें |
| POST | /contacts | संपर्क बनाएं |
| PUT | /contacts/:id | संपर्क अपडेट करें |
| DELETE | /contacts/:id | संपर्क हटाएं |

### समूह

| विधि | एंडपॉइंट | विवरण |
|--------|----------|-------------|
| GET | /groups | समूह सूचीबद्ध करें |
| GET | /groups/:id | समूह प्राप्त करें |
| POST | /groups | समूह बनाएं |
| PUT | /groups/:id | समूह अपडेट करें |
| DELETE | /groups/:id | समूह हटाएं |
| POST | /groups/:id/contacts | संपर्क जोड़ें |

### अभियान

| विधि | एंडपॉइंट | विवरण |
|--------|----------|-------------|
| GET | /campaigns | अभियान सूचीबद्ध करें |
| GET | /campaigns/:id | अभियान प्राप्त करें |
| POST | /campaigns | अभियान बनाएं |
| PUT | /campaigns/:id | अभियान अपडेट करें |
| POST | /campaigns/:id/start | प्रारंभ करें |
| POST | /campaigns/:id/pause | रोकें |
| POST | /campaigns/:id/contacts | संपर्क जोड़ें |
| GET | /campaigns/:id/stats | आंकड़े |

### Webhooks

| विधि | एंडपॉइंट | विवरण |
|--------|----------|-------------|
| GET | /webhooks | webhooks सूचीबद्ध करें |
| POST | /webhooks | webhook बनाएं |
| DELETE | /webhooks/:id | हटाएं |

## SDK और लाइब्रेरी

### JavaScript/TypeScript

```bash
npm install @fluenzr/sdk
```

```javascript
import { FluenzrClient } from '@fluenzr/sdk';

const client = new FluenzrClient({
  apiKey: 'your-api-key'
});

const contacts = await client.contacts.list();
```

### Python

```bash
pip install fluenzr
```

```python
from fluenzr import FluenzrClient

client = FluenzrClient(api_key='your-api-key')
contacts = client.contacts.list()
```

## उपयोग उदाहरण

### CRM सिंक करें

```javascript
// अपने CRM से नए संपर्क प्राप्त करें
const crmContacts = await myCRM.getNewContacts();

// उन्हें FluenzR में बनाएं
for (const contact of crmContacts) {
  await fluenzr.contacts.create({
    email: contact.email,
    firstName: contact.first_name,
    lastName: contact.last_name,
    company: contact.company
  });
}
```

### अभियान स्वचालित करें

```javascript
// API के माध्यम से अभियान बनाएं
const campaign = await fluenzr.campaigns.create({
  name: 'Automated Q1 2025',
  emailAccountId: 'acc_123',
  // ...
});

// संपर्क जोड़ें
await fluenzr.campaigns.addContacts(campaign.id, {
  groupId: 'grp_456'
});

// प्रारंभ करें
await fluenzr.campaigns.start(campaign.id);
```

### आंकड़े निर्यात करें

```javascript
// सभी अभियानों के लिए आंकड़े प्राप्त करें
const campaigns = await fluenzr.campaigns.list();

for (const campaign of campaigns) {
  const stats = await fluenzr.campaigns.getStats(campaign.id);
  console.log(`${campaign.name}: ${stats.openRate}% opens, ${stats.replyRate}% replies`);
}
```
