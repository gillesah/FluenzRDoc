# अभियान API

अभियान बनाने, नियंत्रित करने और विश्लेषण करने के लिए एंडपॉइंट।

```json

  "status": "active",
  "emailAccountId": "acc_xyz789",
```

## अभियान सूचीबद्ध करें

अभियानों की सूची प्राप्त करता है।

```
GET /campaigns
```

### क्वेरी पैरामीटर

| पैरामीटर | प्रकार | विवरण |
|-----------|------|-------------|
| `page` | integer | पृष्ठ संख्या |
| `perPage` | integer | प्रति पृष्ठ आइटम |
| `status` | string | स्थिति द्वारा फ़िल्टर |
| `sort` | string | सॉर्ट फ़ील्ड |
| `order` | string | asc या desc |

### संभावित स्थितियां

| स्थिति | विवरण |
|--------|-------------|
| `draft` | ड्राफ्ट |
| `active` | चल रहा है |
| `paused` | रोका गया |
| `completed` | पूर्ण |
| `archived` | संग्रहीत |

### उदाहरण

```bash
curl -X GET "https://api.fluenzr.co/v1/campaigns?status=active" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### प्रतिक्रिया

```json
{
  "success": true,
  "data": [
    {
      "id": "cmp_abc123",
      "name": "Q1 2025 प्रॉस्पेक्टिंग",
      "status": "active",
      ...
    }
  ],
  "meta": {
    "page": 1,
    "perPage": 20,
    "total": 15
  }
}
```

## एक अभियान प्राप्त करें

```
GET /campaigns/:id
```

### उदाहरण

```bash
curl -X GET "https://api.fluenzr.co/v1/campaigns/cmp_abc123" \
  -H "Authorization: Bearer flz_sk_xxx"
```

## अभियान बनाएं

```
POST /campaigns
```

### अनुरोध बॉडी

| फ़ील्ड | प्रकार | आवश्यक | विवरण |
|-------|------|----------|-------------|
| `name` | string | हां | अभियान नाम |
| `description` | string | | विवरण |
| `emailAccountId` | string | हां | ईमेल खाता ID |
| `settings` | object | | भेजने की सेटिंग्स |
| `workflow` | object | | वर्कफ़्लो संरचना |

### सेटिंग्स संरचना

```json
{
  "sendDays": ["monday", "tuesday", "wednesday", "thursday", "friday"],
  "sendHoursStart": 9,
  "sendHoursEnd": 18,
  "timezone": "Asia/Kolkata",
  "maxEmailsPerHour": 30
}
```

### वर्कफ़्लो संरचना

```json
{
  "nodes": [
    {
      "id": "node_1",
      "type": "start"
    },
    {
      "id": "node_2",
      "type": "email",
      "data": {
        "subject": "{{firstName}}, त्वरित प्रश्न",
        "body": "नमस्ते {{firstName}},\n\nमैं आपसे संपर्क कर रहा हूं...",
        "trackOpens": true,
        "trackClicks": true
      }
    },
    {
      "id": "node_3",
      "type": "delay",
      "data": {
        "days": 3,
        "businessDays": true
      }
    },
    {
      "id": "node_4",
      "type": "email",
      "data": {
        "subject": "मेरे संदेश पर फॉलो-अप",
        "body": "{{firstName}},\n\nमैं फॉलो-अप कर रहा हूं..."
      }
    },
    {
      "id": "node_5",
      "type": "end"
    }
  ],
  "edges": [
    {"from": "node_1", "to": "node_2"},
    {"from": "node_2", "to": "node_3"},
    {"from": "node_3", "to": "node_4"},
    {"from": "node_4", "to": "node_5"}
  ]
}
```

### पूर्ण उदाहरण

```bash
curl -X POST "https://api.fluenzr.co/v1/campaigns" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Q1 2025 प्रॉस्पेक्टिंग",
    "emailAccountId": "acc_xyz789",
    "settings": {
      "sendDays": ["monday", "tuesday", "wednesday", "thursday", "friday"],
      "sendHoursStart": 9,
      "sendHoursEnd": 18,
      "timezone": "Asia/Kolkata"
    },
    "workflow": {
      "nodes": [...],
      "edges": [...]
    }
  }'
```

## अभियान अपडेट करें

```
PUT /campaigns/:id
```

> **चेतावनी**
केवल `draft` या `paused` स्थिति वाले अभियानों को संशोधित किया जा सकता है।

### उदाहरण

```bash
curl -X PUT "https://api.fluenzr.co/v1/campaigns/cmp_abc123" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Q1 2025 प्रॉस्पेक्टिंग - v2",
    "settings": {
      "maxEmailsPerHour": 50
    }
  }'
```

## अभियान नियंत्रित करें

### प्रारंभ करें

एक ड्राफ्ट अभियान लॉन्च करता है।

```
POST /campaigns/:id/start
```

```bash
curl -X POST "https://api.fluenzr.co/v1/campaigns/cmp_abc123/start" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### रोकें

एक सक्रिय अभियान को निलंबित करता है।

```
POST /campaigns/:id/pause
```

### फिर से शुरू करें

एक रोके गए अभियान को पुनरारंभ करता है।

```
POST /campaigns/:id/resume
```

### संग्रहित करें

एक पूर्ण अभियान को संग्रहित करता है।

```
POST /campaigns/:id/archive
```

## अभियान संपर्क प्रबंधित करें

### संपर्क जोड़ें

```
POST /campaigns/:id/contacts
```

```json
{
  "contactIds": ["con_abc", "con_def"],
  // या
  "groupIds": ["grp_xyz"],
  // या
  "filters": {
    "status": "active",
    "customFields": {
      "sector": "Fintech"
    }
  }
}
```

### उदाहरण

```bash
curl -X POST "https://api.fluenzr.co/v1/campaigns/cmp_abc123/contacts" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "groupIds": ["grp_xyz789"]
  }'
```

### प्रतिक्रिया

```json
{
  "success": true,
  "data": {
    "added": 150,
    "skipped": 5,
    "errors": []
  }
}
```

### संपर्क हटाएं

```
DELETE /campaigns/:id/contacts
```

```json
{
  "contactIds": ["con_abc", "con_def"]
}
```

### संपर्क सूचीबद्ध करें

```
GET /campaigns/:id/contacts
```

| पैरामीटर | विवरण |
|-----------|-------------|
| `status` | स्थिति द्वारा फ़िल्टर (pending, in_progress, completed, replied, bounced) |
| `page` | पृष्ठ |
| `perPage` | प्रति पृष्ठ आइटम |

### टाइमलाइन सांख्यिकी

```
GET /campaigns/:id/stats/timeline
```

| पैरामीटर | विवरण |
|-----------|-------------|
| `from` | प्रारंभ तिथि (ISO 8601) |
| `to` | समाप्ति तिथि (ISO 8601) |
| `interval` | ग्रैन्युलैरिटी: hour, day, week |

## विस्तृत गतिविधि

### इवेंट सूची

```
GET /campaigns/:id/activity
```

| पैरामीटर | विवरण |
|-----------|-------------|
| `type` | प्रकार द्वारा फ़िल्टर (sent, opened, clicked, replied, bounced) |
| `contactId` | संपर्क द्वारा फ़िल्टर |
| `from` | प्रारंभ तिथि |
| `to` | समाप्ति तिथि |
