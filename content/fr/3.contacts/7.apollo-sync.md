# Synchronisation Apollo.io

FluenzR s'intègre avec Apollo.io pour vous permettre d'importer et synchroniser vos contacts de prospection facilement.

## Prérequis

- Un compte Apollo.io avec une clé API
- Au moins un contact dans Apollo.io

## Configuration

### 1. Obtenir votre clé API Apollo

1. Connectez-vous à votre compte Apollo.io
2. Allez dans **Settings** > **Integrations** > **API Keys**
3. Créez une nouvelle clé API ou copiez une clé existante

### 2. Configurer FluenzR

1. Dans FluenzR, allez dans **Intégrations**
2. Onglet **Enrichissement (Apollo/Hunter)**
3. Collez votre clé API Apollo
4. Cliquez sur **Enregistrer**

## Importer des contacts

### 1. Accéder à Apollo Sync

1. Allez dans **Intégrations**
2. Cliquez sur l'onglet **Apollo Sync**

### 2. Sélectionner une liste

FluenzR affiche vos listes Apollo disponibles :

- **Listes avec badge "Importé"** : Listes déjà synchronisées avec FluenzR
- **Listes sans badge** : Listes non encore importées
- **"Tous les contacts"** : Importer tous vos contacts Apollo

### 3. Prévisualisation

Après avoir sélectionné une liste, FluenzR affiche :

| Information | Description |
|-------------|-------------|
| **Nouveaux contacts** | Contacts qui n'existent pas encore dans FluenzR |
| **Doublons détectés** | Contacts avec un email déjà présent dans FluenzR |
| **Total Apollo** | Nombre total de contacts dans la liste |

### 4. Gestion des doublons

Pour chaque doublon détecté, vous pouvez choisir :

| Action | Comportement |
|--------|--------------|
| **Ignorer** | Le contact existant n'est pas modifié |
| **Mettre à jour** | Les champs du contact sont mis à jour avec les données Apollo |

> **Astuce**
> Utilisez les boutons "Tout ignorer" ou "Tout mettre à jour" pour appliquer la même action à tous les doublons.

### 5. Groupe de destination

Choisissez où placer les contacts importés :

| Option | Description |
|--------|-------------|
| **Créer un nouveau groupe** | Un nouveau groupe sera créé avec le nom de la liste Apollo |
| **Groupe existant** | Ajouter les contacts à un groupe existant |
| **Pas de groupe** | Importer les contacts sans les assigner à un groupe |

> **Info**
> Vous pouvez personnaliser le nom du groupe avant l'import.

### 6. Lancer l'import

Cliquez sur **Importer** (ou **Synchroniser** pour les listes déjà importées).

## Synchronisation

### Re-synchroniser une liste

Pour les listes déjà importées :

1. La liste affiche le badge vert **Importé** et le nom du groupe associé
2. Cliquez sur **Synchroniser** au lieu de "Importer"
3. Les nouveaux contacts sont ajoutés, les existants peuvent être mis à jour
4. Le groupe de destination est automatiquement pré-sélectionné

### Modifier le nom du groupe

Lors d'une synchronisation, vous pouvez modifier le nom du groupe existant. Le changement sera appliqué automatiquement.

## Champs synchronisés

Les champs suivants sont importés depuis Apollo :

| Champ Apollo | Champ FluenzR |
|--------------|---------------|
| `email` | Email |
| `first_name` | Prénom |
| `last_name` | Nom |
| `organization.name` | Entreprise |
| `title` | Poste |
| `phone_numbers[0]` | Téléphone |
| `linkedin_url` | URL LinkedIn |
| `city` | Ville |
| `country` | Pays |

## Rapport de synchronisation

À la fin de chaque synchronisation, un rapport affiche :

- Nombre de contacts **créés**
- Nombre de contacts **mis à jour**
- Nombre de contacts **ignorés**
- Erreurs éventuelles (avec détails)

## Limites

| Limite | Valeur |
|--------|--------|
| Contacts par synchronisation | Illimité |
| Listes détectées | 10 premières pages de contacts |

## Résolution de problèmes

### "Clé API invalide"

- Vérifiez que la clé API est correctement copiée
- Assurez-vous que la clé n'a pas expiré dans Apollo

### "Limite de requêtes atteinte"

Apollo limite le nombre de requêtes par minute. Attendez quelques instants et réessayez.

### Contacts sans email

Les contacts Apollo sans email sont quand même importés, mais ne pourront pas recevoir de campagnes email. Utilisez l'enrichissement pour trouver leurs emails.

### Liste vide ou non affichée

FluenzR détecte les listes en analysant les labels de vos contacts. Si une liste n'apparaît pas :

1. Vérifiez que des contacts sont assignés à cette liste dans Apollo
2. Cliquez sur **Rafraîchir** pour recharger les listes

## Bonnes pratiques

### Avant l'import

- Nettoyez vos listes Apollo des contacts obsolètes
- Définissez une stratégie de nommage pour vos groupes FluenzR
- Décidez de votre politique de mise à jour des doublons

### Organisation recommandée

1. **Une liste Apollo = Un groupe FluenzR** pour faciliter le suivi
2. Synchronisez régulièrement pour garder vos contacts à jour
3. Utilisez les groupes pour segmenter vos campagnes

### Après l'import

- Vérifiez quelques contacts au hasard
- Lancez un enrichissement email sur les contacts sans email
- Créez une campagne test vers un petit échantillon
