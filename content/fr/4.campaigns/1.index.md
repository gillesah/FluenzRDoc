# Campagnes

Les campagnes sont le cœur de FluenzR. Créez des séquences d'emails automatisées pour engager vos prospects de manière personnalisée et efficace.

## Qu'est-ce qu'une campagne ?

Une campagne FluenzR est une séquence d'emails automatisée qui :

- Envoie des emails personnalisés à vos contacts
- Respecte des délais entre chaque email
- S'arrête automatiquement si le contact répond
- Suit les ouvertures, clics et réponses

## Composants d'une campagne

### Workflow visuel

L'éditeur de workflow vous permet de créer votre séquence visuellement :

```
[Démarrage]
    ↓
[Email 1] → "Premier contact"
    ↓
[Délai 3 jours]
    ↓
[Email 2] → "Relance"
    ↓
[Délai 5 jours]
    ↓
[Email 3] → "Dernière relance"
    ↓
[Fin]
```

### Types de nœuds

| Nœud | Description |
|------|-------------|
| **Email** | Envoie un email personnalisé |
| **Délai** | Attend X jours/heures avant le prochain nœud |
| **Condition** | Branche selon un critère (Plan Pro) |
| **Fin** | Termine la séquence |

## États d'une campagne

| État | Description |
|------|-------------|
| **Brouillon** | En cours de création, non active |
| **Active** | En cours d'exécution |
| **En pause** | Temporairement suspendue |
| **Terminée** | Tous les contacts ont terminé la séquence |
| **Archivée** | Campagne archivée |

## Cycle de vie d'un contact dans une campagne

1. **Ajouté** : Le contact entre dans la campagne
2. **En cours** : Le contact progresse dans la séquence
3. **Répondu** : Le contact a répondu → Séquence arrêtée
4. **Terminé** : Le contact a reçu tous les emails
5. **Désinscrit** : Le contact s'est désinscrit
6. **Erreur** : Problème d'envoi (bounce, etc.)

## Fonctionnalités

### Personnalisation

- Variables dynamiques : `{{firstName}}`, `{{company}}`, etc.
- Champs personnalisés dans les emails
- Signature personnalisable

### Suivi

- Tracking des ouvertures
- Tracking des clics
- Détection des réponses
- Statistiques en temps réel

### Optimisation (Starter+)

- A/B testing sur les objets
- A/B testing sur le contenu
- Analytics avancés

### Automatisation avancée (Pro)

- Workflows conditionnels
- Webhooks sortants
- Intégration API

## Limites par plan

| Plan | Emails/jour | A/B Testing | Workflows conditionnels |
|------|-------------|-------------|------------------------|
| Free | 50 | Non | Non |
| Starter | Illimité* | Oui | Non |
| Pro | Illimité* | Oui | Oui |

*Selon les limites de votre fournisseur email
