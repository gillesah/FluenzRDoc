# SMTP設定

GmailとOutlook以外のメールプロバイダーの場合、FluenzRはSMTP経由での接続をサポートしています。

## いつSMTPを使用するか？

- セルフホスト型メールサーバー
- フランスのメールプロバイダー(OVH、Gandiなど)
- プロフェッショナルメールサービス(Zoho、ProtonMail Bridgeなど)
- 特定の設定ニーズ

> **警告**
SMTP接続は、認証情報を暗号化して保存します(AES-256)。最大限のセキュリティを確保するには、可能な限りOAuth(Gmail/Outlook)を使用してください。

## 必要な情報

SMTPを設定するには、以下が必要です:

| パラメータ | 説明 | 例 |
|-----------|-------------|---------|
| **SMTPサーバー** | 送信サーバーのアドレス | smtp.example.com |
| **ポート** | 接続ポート | 587 (TLS) または 465 (SSL) |
| **ユーザー名** | 通常はメールアドレス | user@example.com |
| **パスワード** | パスワードまたはアプリパスワード | ********** |
| **暗号化** | セキュリティタイプ | TLS (推奨) |

## ステップバイステップの設定

### 1. 設定にアクセス

1. **メールアカウント**に移動します
2. **+ SMTP**をクリックします

### 2. 情報を入力

1. **メール**: 完全なメールアドレス
2. **表示名**: 送信者として表示される名前
3. **SMTPサーバー**: サーバーのアドレス
4. **ポート**: 通常587 (TLS) または 465 (SSL)
5. **ユーザー名**: メールアドレスと同じことが多い
6. **パスワード**: パスワード(またはアプリパスワード)
7. **暗号化**: TLSまたはSSLを選択

### 3. 接続をテスト

**テスト**をクリックして、設定が機能することを確認します。テストメールが自分のアドレスに送信されます。

### 4. IMAPの設定(オプション)

返信検出のために、IMAPも設定します:

| パラメータ | 説明 | 例 |
|-----------|-------------|---------|
| **IMAPサーバー** | 受信サーバーのアドレス | imap.example.com |
| **ポート** | 接続ポート | 993 (SSL) |

## 一般的な設定

### OVH Mail

```
SMTP: ssl0.ovh.net
ポート: 587 (TLS)
IMAP: ssl0.ovh.net
IMAPポート: 993 (SSL)
```

### Gandi

```
SMTP: mail.gandi.net
ポート: 587 (TLS)
IMAP: mail.gandi.net
IMAPポート: 993 (SSL)
```

### Zoho Mail

```
SMTP: smtp.zoho.eu (または smtp.zoho.com)
ポート: 587 (TLS)
IMAP: imap.zoho.eu
IMAPポート: 993 (SSL)
```

> **情報**
Zohoの場合、アカウントのセキュリティ設定で「アプリパスワード」を生成する必要があります。

### Infomaniak

```
SMTP: mail.infomaniak.com
ポート: 587 (TLS)
IMAP: mail.infomaniak.com
IMAPポート: 993 (SSL)
```

### ProtonMail (Bridge経由)

```
SMTP: 127.0.0.1
ポート: 1025
IMAP: 127.0.0.1
IMAPポート: 1143
```

> **警告**
ProtonMailは、マシンにBridgeアプリケーションがインストールされ、アクティブになっている必要があります。

## アプリパスワード

一部のプロバイダーでは、メインパスワードの代わりに「アプリパスワード」が必要です:

### アプリパスワードをいつ使用するか？

- 二要素認証(2FA)が有効になっている
- プロバイダーのセキュリティ要件
- 組織のポリシー

### 生成方法は？

1. プロバイダーのWebインターフェースにログインします
2. 設定 > セキュリティに移動します
3. 「アプリパスワード」または「App Passwords」を探します
4. 「FluenzR」用の新しいパスワードを作成します
5. 生成されたパスワードをFluenzRにコピーします

## トラブルシューティング

### 「接続が拒否されました」

- ポートとサーバーを確認してください
- ファイアウォールがブロックしていないか確認してください
- 別のポートでテストしてください(587の代わりに465)

### 「認証に失敗しました」

- ユーザー名を確認してください(多くの場合、完全なメールアドレス)
- 2FAが有効な場合はアプリパスワードを使用してください
- パスワードの大文字小文字を確認してください

### 「証明書が無効です」

- 正しい暗号化タイプを使用していることを確認してください
- 証明書が期限切れの場合はプロバイダーに連絡してください

### 「送信制限に達しました」

SMTPサーバーには多くの場合制限があります:
- プロバイダーの制限を確認してください
- 送信の間隔を空けてください
- クォータを増やすためにホストに連絡してください

## セキュリティ

### 認証情報の保存

- SMTP認証情報はAES-256で暗号化されます
- データベースに平文で保存されることはありません
- 送信システムからのみアクセス可能です

### 推奨事項

1. 専用のアプリパスワードを使用してください
2. プロバイダーでログインアラートを有効にしてください
3. パスワードを定期的に変更してください
4. 可能な限りOAuthを優先してください
