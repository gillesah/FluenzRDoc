# Connecter Gmail

Gmail est le provider recommandé pour FluenzR grâce à sa fiabilité et sa bonne délivrabilité.

## Prérequis

- Un compte Gmail personnel ou Google Workspace
- Accès à la page des comptes email dans FluenzR

## Étapes de connexion

### 1. Accéder à la configuration

1. Connectez-vous à votre compte FluenzR
2. Dans le menu latéral, cliquez sur **Comptes Email**
3. Cliquez sur le bouton **+ Gmail**

### 2. Autoriser l'accès

1. Une fenêtre Google s'ouvre
2. Sélectionnez le compte Gmail à connecter
3. Cliquez sur **Continuer**

> **⚠️ Attention**
**Application non vérifiée** : Vous verrez un avertissement "Cette application n'est pas validée par Google". C'est normal pendant notre processus de certification. Cliquez sur "Paramètres avancés" puis "Accéder à FluenzR (non sécurisé)".

### 3. Accorder les permissions

FluenzR demande les permissions suivantes :
- **Envoyer des emails** : Pour envoyer vos campagnes
- **Lire les emails** : Pour détecter les réponses
- **Voir votre adresse email** : Pour identifier le compte

Cliquez sur **Autoriser** pour confirmer.

### 4. Vérification

Après l'autorisation, vous êtes redirigé vers FluenzR. Votre compte Gmail apparaît dans la liste avec le statut **Connecté**.

## Limites Gmail

| Type de compte | Limite quotidienne |
|----------------|-------------------|
| Gmail personnel | ~500 emails/jour |
| Google Workspace | ~2000 emails/jour |

> **ℹ️ Info**
Ces limites sont imposées par Google, pas par FluenzR. Les dépasser peut entraîner une suspension temporaire de votre compte.

## Bonnes pratiques

### Commencez progressivement
Ne passez pas de 0 à 500 emails/jour immédiatement. Augmentez progressivement :
- Semaine 1 : 20-30 emails/jour
- Semaine 2 : 50-100 emails/jour
- Semaine 3+ : Augmentez selon les résultats

### Activez le warming
Le [warming](/docs/email-accounts/warming) améliore la réputation de votre compte en simulant des conversations naturelles.

### Utilisez plusieurs comptes (Starter+)
Avec le plan Starter ou Pro, connectez plusieurs comptes Gmail pour :
- Répartir la charge
- Tester différentes approches
- Éviter les blocages

## Résolution de problèmes

### "Échec de la connexion"
- Vérifiez que les pop-ups ne sont pas bloqués
- Essayez avec un autre navigateur
- Désactivez temporairement les extensions

### "Quota dépassé"
- Attendez 24h avant de réessayer
- Réduisez le volume d'envoi
- Ajoutez des comptes supplémentaires

### "Token expiré"
- Allez dans Comptes Email
- Cliquez sur **Reconnecter** à côté du compte concerné
- Refaites le processus d'autorisation

## Révocation d'accès

Pour déconnecter FluenzR de votre compte Gmail :

1. Allez sur [myaccount.google.com/permissions](https://myaccount.google.com/permissions)
2. Trouvez "FluenzR" dans la liste
3. Cliquez sur **Supprimer l'accès**

> **⚠️ Attention**
La révocation côté Google n'efface pas le compte dans FluenzR. Pensez à le supprimer également dans l'application.
