# Pipeline CRM

Le pipeline FluenzR vous permet de suivre visuellement l'avancement de vos prospects dans votre processus de vente.

## Vue d'ensemble

Le pipeline est une vue Kanban qui représente les étapes de votre cycle de vente. Chaque contact peut être positionné dans une étape du pipeline.

### Étapes par défaut

| Étape | Description | Couleur |
|-------|-------------|---------|
| **Lead** | Premier contact, pas encore qualifié | Gris |
| **Qualifié** | Intérêt confirmé, besoin identifié | Bleu |
| **Proposition** | Proposition commerciale envoyée | Jaune |
| **Négociation** | Discussion en cours | Orange |
| **Gagné** | Deal conclu | Vert |
| **Perdu** | Opportunité perdue | Rouge |

## Accéder au pipeline

1. Dans le menu latéral, cliquez sur **Pipeline**
2. La vue Kanban s'affiche avec tous vos contacts positionnés

## Positionner un contact dans le pipeline

### Depuis la vue Pipeline

1. Glissez-déposez le contact d'une colonne à l'autre
2. Le changement est enregistré automatiquement

### Depuis la fiche contact

1. Ouvrez la fiche d'un contact
2. Dans la section "Pipeline", sélectionnez l'étape dans le menu déroulant
3. Le changement est enregistré automatiquement

### Depuis la liste des contacts

1. Dans le tableau des contacts, la colonne "Pipeline" affiche l'étape actuelle
2. Cliquez sur le menu déroulant pour changer l'étape directement

## Personnaliser le pipeline

### Ajouter une étape

1. Dans la vue Pipeline, cliquez sur **+ Ajouter une étape**
2. Définissez le nom et la couleur
3. Positionnez l'étape dans l'ordre souhaité

### Modifier une étape

1. Cliquez sur le menu (⋮) de l'étape
2. Sélectionnez **Modifier**
3. Changez le nom ou la couleur
4. Enregistrez

### Réorganiser les étapes

1. Glissez-déposez l'en-tête de l'étape
2. Repositionnez-la dans l'ordre souhaité

### Supprimer une étape

1. Cliquez sur le menu (⋮) de l'étape
2. Sélectionnez **Supprimer**
3. Choisissez où déplacer les contacts existants
4. Confirmez

> **⚠️ Attention**
Vous ne pouvez pas supprimer une étape qui contient des contacts sans les déplacer d'abord.

## Filtrer le pipeline

### Par groupe

Affichez uniquement les contacts d'un groupe spécifique :
1. Cliquez sur **Filtres**
2. Sélectionnez le groupe

### Par période

Filtrez selon la date d'entrée dans l'étape :
1. Cliquez sur **Filtres**
2. Sélectionnez une plage de dates

### Par propriétaire

Si vous travaillez en équipe (Plan Pro) :
1. Cliquez sur **Filtres**
2. Sélectionnez un membre de l'équipe

## Statistiques du pipeline

### Vue d'ensemble

En haut du pipeline, consultez :
- Nombre total de contacts par étape
- Valeur potentielle (si champ "Valeur deal" renseigné)
- Taux de conversion entre étapes

### Métriques clés

| Métrique | Description |
|----------|-------------|
| **Conversion Lead → Qualifié** | % de leads qualifiés |
| **Temps moyen par étape** | Durée moyenne avant passage à l'étape suivante |
| **Taux de closing** | % de deals gagnés |
| **Valeur moyenne** | Valeur moyenne des deals gagnés |

## Intégration avec les campagnes

### Automatisations possibles

Le pipeline s'intègre avec vos campagnes :

- **Déclencheur** : Contact ajouté à une campagne → Passe en "Lead"
- **Réponse détectée** : → Passe en "Qualifié"
- **Action manuelle** : Envoi de proposition → Passe en "Proposition"

### Suivi des interactions

Dans la fiche contact, consultez l'historique complet :
- Emails envoyés via les campagnes
- Ouvertures et clics
- Réponses reçues
- Changements d'étape

## Bonnes pratiques

### Définir vos étapes

Adaptez les étapes à votre processus réel :

**Exemple B2B SaaS :**
1. Lead
2. Démo planifiée
3. Démo effectuée
4. Proposition
5. Closing
6. Gagné / Perdu

**Exemple Agence :**
1. Prospect
2. Premier appel
3. Brief envoyé
4. Devis
5. Négociation
6. Signé / Perdu

### Maintenir le pipeline à jour

- Déplacez les contacts dès qu'ils progressent
- Revoyez régulièrement les contacts "bloqués"
- Nettoyez les opportunités perdues

### Utiliser les notes

Ajoutez des notes à chaque changement d'étape :
- Raison du passage
- Prochaine action prévue
- Points d'attention

## Travail en équipe (Plan Pro)

### Attribution des contacts

Assignez des contacts à des membres de l'équipe :
1. Ouvrez la fiche du contact
2. Sélectionnez le propriétaire

### Vue par membre

Filtrez le pipeline par membre pour voir :
- Les opportunités de chacun
- La charge de travail
- Les performances individuelles

### Permissions

| Rôle | Permissions pipeline |
|------|---------------------|
| Admin | Tout (config + données) |
| Membre | Voir et modifier ses contacts |
