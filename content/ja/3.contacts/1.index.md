# 連絡先管理

連絡先はキャンペーンの中心です。FluenzRは、連絡先をインポート、整理、セグメント化するための強力なツールを提供します。

## 概要

FluenzRの連絡先には以下が含まれます:

| フィールド | 説明 | 必須 |
|-------|-------------|----------|
| **メール** | 一意のメールアドレス | はい |
| **名** | 連絡先の名 | いいえ |
| **姓** | 家族名 | いいえ |
| **会社** | 会社名 | いいえ |
| **役職** | 役職/肩書き | いいえ |
| **電話** | 電話番号 | いいえ |
| **カスタムフィールド** | 追加データ | いいえ |

## プランごとの制限

| プラン | 連絡先の制限 |
|------|---------------|
| Free | 300連絡先 |
| Starter | 無制限 |
| Pro | 無制限 |

## 機能

### 整理

- **グループ**: テーマ、ソース、または基準で連絡先をグループ化します
- **タグ**: 簡単にフィルタリングするためのラベルを追加します
- **カスタムフィールド**: 独自の属性を作成します

### セグメンテーション

- グループ、タグ、またはフィールド値でフィルタリング
- 複数の基準を組み合わせる
- セグメントをエクスポート

### パイプライン

CRMパイプラインで見込み客の進捗を追跡します:
- リード → 認定済み → 提案 → 交渉 → 成約/不成約

### インポート/エクスポート

- 自動マッピング付きCSVインポート
- すべての連絡先または選択した連絡先のエクスポート
- 重複検出

## 連絡先のステータス

| ステータス | 説明 |
|--------|-------------|
| **アクティブ** | キャンペーンで利用可能な連絡先 |
| **配信停止** | 配信停止リンクをクリックした |
| **バウンス** | 無効なメール(バウンス) |
| **返信済み** | キャンペーンに返信した |
| **ブロック** | 手動でブロックされた |

## ベストプラクティス

### データ品質

- インポート前にメールを確認する
- 検証サービスを使用する(Hunter、Neverbounce)
- バウンスを定期的にクリーンアップする

### GDPR準拠

- 必要に応じて同意を収集する
- 配信停止リクエストを尊重する
- 収集ソースを文書化する

### 整理

- 最初から論理的なグループを作成する
- 明確な命名規則を使用する
- 古い連絡先をアーカイブする
