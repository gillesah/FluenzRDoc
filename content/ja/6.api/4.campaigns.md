## キャンペーン一覧

キャンペーン一覧を取得します。

```
GET /campaigns
```

### クエリパラメータ

| パラメータ | 型 | 説明 |
|-----------|------|-------------|
| `page` | integer | ページ番号 |
| `perPage` | integer | ページあたりの件数 |
| `status` | string | ステータスでフィルタリング |
| `sort` | string | ソートフィールド |
| `order` | string | ascまたはdesc |

### 利用可能なステータス

| ステータス | 説明 |
|--------|-------------|
| `draft` | 下書き |
| `active` | 実行中 |
| `paused` | 一時停止 |
| `completed` | 完了 |
| `archived` | アーカイブ済み |

### 例

```bash
curl -X GET "https://api.fluenzr.co/v1/campaigns?status=active" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### レスポンス

```json
{
  "success": true,
  "data": [
    {
      "id": "cmp_abc123",
      "name": "Q1 2025プロスペクティング",
      "status": "active",
      ...
    }
  ],
  "meta": {
    "page": 1,
    "perPage": 20,
    "total": 15
  }
}
```

## キャンペーンの取得

```
GET /campaigns/:id
```

### 例

```bash
curl -X GET "https://api.fluenzr.co/v1/campaigns/cmp_abc123" \
  -H "Authorization: Bearer flz_sk_xxx"
```

## キャンペーンの作成

```
POST /campaigns
```

### リクエストボディ

| フィールド | 型 | 必須 | 説明 |
|-------|------|--------|-------------|
| `name` | string | ✅ | キャンペーン名 |
| `description` | string | | 説明 |
| `emailAccountId` | string | ✅ | メールアカウントID |
| `settings` | object | | 送信設定 |
| `workflow` | object | | ワークフロー構造 |

### settings構造

```json
{
  "sendDays": ["monday", "tuesday", "wednesday", "thursday", "friday"],
  "sendHoursStart": 9,
  "sendHoursEnd": 18,
  "timezone": "Asia/Tokyo",
  "maxEmailsPerHour": 30
}
```

### workflow構造

```json
{
  "nodes": [
    {
      "id": "node_1",
      "type": "start"
    },
    {
      "id": "node_2",
      "type": "email",
      "data": {
        "subject": "{{firstName}}さん、簡単な質問",
        "body": "{{firstName}}様\n\nお忙しいところ失礼します...",
        "trackOpens": true,
        "trackClicks": true
      }
    },
    {
      "id": "node_3",
      "type": "delay",
      "data": {
        "days": 3,
        "businessDays": true
      }
    },
    {
      "id": "node_4",
      "type": "email",
      "data": {
        "subject": "先日のメールの件",
        "body": "{{firstName}}様\n\n再度ご連絡いたします..."
      }
    },
    {
      "id": "node_5",
      "type": "end"
    }
  ],
  "edges": [
    {"from": "node_1", "to": "node_2"},
    {"from": "node_2", "to": "node_3"},
    {"from": "node_3", "to": "node_4"},
    {"from": "node_4", "to": "node_5"}
  ]
}
```

### 完全な例

```bash
curl -X POST "https://api.fluenzr.co/v1/campaigns" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Q1 2025プロスペクティング",
    "emailAccountId": "acc_xyz789",
    "settings": {
      "sendDays": ["monday", "tuesday", "wednesday", "thursday", "friday"],
      "sendHoursStart": 9,
      "sendHoursEnd": 18,
      "timezone": "Asia/Tokyo"
    },
    "workflow": {
      "nodes": [...],
      "edges": [...]
    }
  }'
```

## キャンペーンの更新

```
PUT /campaigns/:id
```

> **⚠️ 注意**
`draft`または`paused`ステータスのキャンペーンのみ編集可能です。

### 例

```bash
curl -X PUT "https://api.fluenzr.co/v1/campaigns/cmp_abc123" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Q1 2025プロスペクティング - v2",
    "settings": {
      "maxEmailsPerHour": 50
    }
  }'
```

## キャンペーンの制御

### 開始

下書きのキャンペーンを開始します。

```
POST /campaigns/:id/start
```

```bash
curl -X POST "https://api.fluenzr.co/v1/campaigns/cmp_abc123/start" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### 一時停止

アクティブなキャンペーンを一時停止します。

```
POST /campaigns/:id/pause
```

### 再開

一時停止中のキャンペーンを再開します。

```
POST /campaigns/:id/resume
```

### アーカイブ

完了したキャンペーンをアーカイブします。

```
POST /campaigns/:id/archive
```

## キャンペーンの連絡先管理

### 連絡先の追加

```
POST /campaigns/:id/contacts
```

```json
{
  "contactIds": ["con_abc", "con_def"],
  // または
  "groupIds": ["grp_xyz"],
  // または
  "filters": {
    "status": "active",
    "customFields": {
      "sector": "Fintech"
    }
  }
}
```

### 例

```bash
curl -X POST "https://api.fluenzr.co/v1/campaigns/cmp_abc123/contacts" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "groupIds": ["grp_xyz789"]
  }'
```

### レスポンス

```json
{
  "success": true,
  "data": {
    "added": 150,
    "skipped": 5,
    "errors": []
  }
}
```

### 連絡先の削除

```
DELETE /campaigns/:id/contacts
```

```json
{
  "contactIds": ["con_abc", "con_def"]
}
```

### 連絡先一覧

```
GET /campaigns/:id/contacts
```

| パラメータ | 説明 |
|-----------|-------------|
| `status` | ステータスでフィルタリング（pending、in_progress、completed、replied、bounced） |
| `page` | ページ |
| `perPage` | ページあたりの件数 |
