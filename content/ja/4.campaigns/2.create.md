# キャンペーンを作成

このガイドでは、FluenzRで初めてのキャンペーンを作成する方法を説明します。

## ステップ1: 作成画面へアクセス

1. サイドメニューで**キャンペーン**をクリック
2. **+ 新規キャンペーン**ボタンをクリック

## ステップ2: 基本設定

### 一般情報

| フィールド | 説明 | 例 |
|-------|-------------|---------|
| **名前** | キャンペーンの内部識別名 | "2025年Q1プロスペクティング - Fintech" |
| **説明** | 説明メモ（任意） | "LinkedInリード向け3メールシーケンス" |

### メールアカウント

メッセージを送信するメールアカウントを選択：
- 複数のアカウントがある場合、自動ローテーションが有効化（Starter以上）
- アカウントが接続済みでアクティブであることを確認

### 送信設定

| パラメータ | 説明 | 推奨 |
|-----------|-------------|----------------|
| **送信曜日** | 送信可能な曜日 | 月-金 |
| **送信時間** | 時間帯 | 9時-18時 |
| **タイムゾーン** | 時間の基準 | Asia/Tokyo |
| **最大メール数/時間** | 1時間あたりの上限 | 20-30 |

> **情報**
タイミング設定により、最適な時間にメールを送信し、スパムフィルターを回避できます。

## ステップ3: ワークフローを作成

ワークフローエディターが自動的に開きます。

### 最初のメールを追加

1. **開始**ノードは既に配置されています
2. ツールバーで**+ メール**をクリック
3. メールノードを開始に接続

### メールを設定

メールノードをクリックして編集：

#### 件名

```
{{firstName}}さん、{{company}}について簡単な質問
```

#### 本文

```
{{firstName}}様

{{company}}が{{secteur}}分野で事業を展開されているのを拝見し、
ご連絡させていただきました。

FluenzRでは、御社のような企業様の営業活動を、
パーソナライズされたアプローチを維持しながら
自動化するお手伝いをしています。

今週、15分ほどお電話でお話しできますでしょうか？

よろしくお願いいたします。
[署名]
```

#### オプション

| オプション | 説明 |
|--------|-------------|
| **開封トラッキング** | 開封追跡を有効化 |
| **クリックトラッキング** | クリック追跡を有効化 |

### 待機を追加

1. **+ 待機**をクリック
2. 期間を設定（例：3日）
3. 最初のメールの後に接続

### フォローアップメールを追加

シーケンスを作成するプロセスを繰り返します：

```
メール1 → 3日待機 → メール2 → 5日待機 → メール3
```

### ワークフローを完成

**終了**ノードを追加してシーケンスの終了をマーク。

## ステップ4: 連絡先を追加

### 方法1: グループから

1. **連絡先を追加**をクリック
2. **グループ**タブを選択
3. 希望のグループをチェック
4. **追加**をクリック

### 方法2: 個別選択

1. **連絡先を追加**をクリック
2. **連絡先**タブを選択
3. 必要に応じてフィルターを使用
4. 希望の連絡先をチェック
5. **追加**をクリック

### 確認

続行前に確認：
- 追加された連絡先数
- 他のアクティブなキャンペーンと重複がないこと
- すべてが有効なメールアドレスを持っていること

## ステップ5: プレビュー

開始前にメールをプレビュー：

1. **プレビュー**をクリック
2. サンプル連絡先を選択
3. 変数が正しく置換されていることを確認
4. メールのレンダリングを確認

> **注意**
受信トレイでのレンダリングを確認するため、自分のアドレスにテストメールを送信してください。

## ステップ6: キャンペーンを開始

1. **キャンペーンを開始**をクリック
2. 概要を確認：
   - 連絡先数
   - シーケンス内のメール数
   - 送信設定
3. **確認**をクリック

## 開始後

### リアルタイム追跡

- 最初のメールは設定に従って送信開始
- **統計**タブでメトリクスを確認
- 返信は**受信トレイ**に表示

### 可能なアクション

| アクション | 説明 |
|--------|-------------|
| **一時停止** | 送信を一時的に停止 |
| **再開** | 一時停止したキャンペーンを再開 |
| **連絡先を追加** | 新しい連絡先を追加 |
| **アーカイブ** | 完了したキャンペーンをアーカイブ |

## 効果的なキャンペーンのためのヒント

### メールの件名

- 名前や会社名でパーソナライズ
- スパムワードを避ける（無料、緊急など）
- 短く保つ（50文字未満）
- A/Bテストで検証

### コンテンツ

- 要点を簡潔に
- 1通のメールに1つの質問/アクション
- 最大限にパーソナライズ
- 大きな添付ファイルを避ける

### タイミング

- 適切な間隔でフォローアップ（3-5日）
- シーケンスは最大3-4通に制限
- 営業時間内に送信

### 署名

- 連絡先情報を含める
- 適切な場合はLinkedInの写真を追加
- シンプルでプロフェッショナルな署名を維持
