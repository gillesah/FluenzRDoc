# キャンペーン

キャンペーンはFluenzRの中核です。見込み客にパーソナライズされた効果的なアプローチを行うための自動メールシーケンスを作成できます。

## キャンペーンとは？

FluenzRキャンペーンは以下を実現する自動メールシーケンスです：

- 連絡先にパーソナライズされたメールを送信
- 各メール間に適切な間隔を設定
- 連絡先が返信した場合は自動的に停止
- 開封、クリック、返信を追跡

## キャンペーンの構成要素

### ビジュアルワークフロー

ワークフローエディターでシーケンスを視覚的に作成できます：

```
[開始]
    ↓
[メール1] → "初回コンタクト"
    ↓
[3日間待機]
    ↓
[メール2] → "フォローアップ"
    ↓
[5日間待機]
    ↓
[メール3] → "最終フォローアップ"
    ↓
[終了]
```

### ノードの種類

| ノード | 説明 |
|------|-------------|
| **メール** | パーソナライズされたメールを送信 |
| **待機** | 次のノードまでX日/時間待機 |
| **条件** | 条件に応じて分岐（Proプラン） |
| **終了** | シーケンスを終了 |

## キャンペーンの状態

| 状態 | 説明 |
|------|-------------|
| **下書き** | 作成中、未アクティブ |
| **アクティブ** | 実行中 |
| **一時停止** | 一時的に停止中 |
| **完了** | すべての連絡先がシーケンスを完了 |
| **アーカイブ済み** | アーカイブされたキャンペーン |

## キャンペーン内の連絡先ライフサイクル

1. **追加済み**: 連絡先がキャンペーンに登録
2. **進行中**: 連絡先がシーケンスを進行中
3. **返信済み**: 連絡先が返信 → シーケンス停止
4. **完了**: 連絡先がすべてのメールを受信
5. **配信停止**: 連絡先が配信停止
6. **エラー**: 送信問題（バウンスなど）

## 機能

### パーソナライゼーション

- 動的変数: `{{firstName}}`、`{{company}}`など
- メール内のカスタムフィールド
- カスタマイズ可能な署名

### トラッキング

- 開封追跡
- クリック追跡
- 返信検出
- リアルタイム統計

### 最適化（Starter以上）

- 件名のA/Bテスト
- コンテンツのA/Bテスト
- 高度な分析

### 高度な自動化（Pro）

- 条件分岐ワークフロー
- Webhook送信
- API連携

## プラン別の制限

| プラン | メール/日 | A/Bテスト | 条件分岐ワークフロー |
|------|-------------|-------------|------------------------|
| Free | 50 | なし | なし |
| Starter | 無制限* | あり | なし |
| Pro | 無制限* | あり | あり |

*メールプロバイダーの制限に依存
