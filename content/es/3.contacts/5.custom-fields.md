# Campos personalizados

Los campos personalizados te permiten añadir información específica a tus contactos más allá de los campos estándar.

## ¿Por qué usar campos personalizados?

### Personalización avanzada

Usa variables personalizadas en tus emails:
- `{{sector}}` → "Fintech"
- `{{companySize}}` → "50-100 empleados"
- `{{contactSource}}` → "LinkedIn Sales Navigator"

### Segmentación

Filtra y segmenta tus contactos según tus criterios de negocio.

### Reporting

Analiza tu rendimiento según dimensiones personalizadas.

## Crear un campo personalizado

### 1. Acceder a configuración

1. Ve a **Configuración**
2. Haz clic en **Campos personalizados**
3. Haz clic en **+ Nuevo campo**

### 2. Configurar el campo

| Parámetro | Descripción | Ejemplo |
|-----------|-------------|---------|
| **Nombre** | Identificador técnico (sin espacios) | `industry_sector` |
| **Etiqueta** | Nombre de visualización | "Sector Industrial" |
| **Tipo** | Tipo de datos | Texto, Número, Fecha, Lista |
| **Obligatorio** | ¿Campo requerido? | No |
| **Valor por defecto** | Valor si no se proporciona | - |

### 3. Tipos de campo disponibles

| Tipo | Descripción | Caso de uso |
|------|-------------|----------|
| **Texto** | Cadena de texto libre | Nota, comentario |
| **Número** | Valor numérico | Ingresos, número de empleados |
| **Fecha** | Fecha (día/mes/año) | Fecha de primer contacto |
| **Lista desplegable** | Elección entre opciones definidas | Sector, fuente |
| **Casilla** | Sí/No | RGPD aceptado |
| **URL** | Enlace web | Perfil de LinkedIn |

## Ejemplos de campos útiles

### Para prospección B2B

| Campo | Tipo | Variable |
|-------|------|----------|
| Sector industrial | Lista | `{{sector}}` |
| Tamaño de empresa | Lista | `{{companySize}}` |
| Ingresos anuales | Número | `{{revenue}}` |
| Fuente del lead | Lista | `{{source}}` |
| Perfil LinkedIn | URL | `{{linkedinUrl}}` |

### Para seguimiento de ventas

| Campo | Tipo | Variable |
|-------|------|----------|
| Fecha primer contacto | Fecha | `{{contactDate}}` |
| Puntuación de calificación | Número | `{{score}}` |
| Interés en producto | Lista | `{{interestedProduct}}` |
| Presupuesto estimado | Número | `{{budget}}` |

| Campo | Tipo | Variable |
|-------|------|----------|

| Fuente de recopilación | Texto | `{{collectionSource}}` |

## Usar campos personalizados

### En formularios

Los campos personalizados aparecen automáticamente en:
- El formulario de creación de contacto
- El formulario de edición de contacto
- Importación CSV (mapeo de columnas)

### En emails

Usa la sintaxis `{{nombre_del_campo}}` en tus plantillas:

```
Hola {{firstName}},

Como experto en {{sector}}, sabes que...

Con un equipo de {{companySize}} personas,
seguramente necesitas...
```

### En filtros

1. Ve a **Contactos**
2. Haz clic en **Filtros**
3. Selecciona el campo personalizado
4. Define la condición (igual a, contiene, etc.)

## Importar con campos personalizados

### Formato CSV

Nombra tus columnas con el nombre técnico del campo:

```csv
email,firstName,lastName,industry_sector,company_size
john@example.com,John,Smith,Fintech,50-100
mary@example.com,Mary,Johnson,SaaS,10-50
```

### Mapeo

Durante la importación, FluenzR detecta automáticamente las columnas correspondientes a tus campos personalizados.

## Editar un campo personalizado

> **Advertencia**
Editar un campo puede impactar los datos existentes. Procede con precaución.

### Lo que puedes editar

- Etiqueta (nombre de visualización)
- Valor por defecto
- Opciones de lista desplegable (solo añadir)

### Lo que no puedes editar

- Nombre técnico (identificador)
- Tipo de campo

Para cambiar el tipo, crea un nuevo campo y migra los datos.

## Eliminar un campo personalizado

1. Ve a **Configuración** > **Campos personalizados**
2. Haz clic en el icono de eliminar
3. Confirma

> **Peligro**
La eliminación es irreversible. Todos los valores de este campo se perderán para todos los contactos.

## Mejores prácticas

### Nomenclatura

| Recomendado | Evitar |
|-------------|-------|
| `industry_sector` | `Industry Sector` |
| `company_size` | `size` |
| `first_contact_date` | `date1` |

### Organización

- Limita el número de campos (máximo 10-15)
- Prefiere listas sobre campos de texto libre
- Documenta el uso de cada campo
- Revisa regularmente los campos no utilizados

### Valores por defecto

- Usa valores por defecto relevantes
- "No especificado" en lugar de vacío para listas
- Facilita los filtros con valores consistentes
