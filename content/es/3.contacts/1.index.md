# Gestion des contacts

Les contacts sont au cœur de vos campagnes. FluenzR vous offre des outils puissants pour les importer, organiser et segmenter.

## Vue d'ensemble

Un contact dans FluenzR contient :

| Champ | Description | Obligatoire |
|-------|-------------|-------------|
| **Email** | Adresse email unique | ✅ |
| **Prénom** | Prénom du contact | Non |
| **Nom** | Nom de famille | Non |
| **Entreprise** | Nom de l'entreprise | Non |
| **Poste** | Fonction/titre | Non |
| **Téléphone** | Numéro de téléphone | Non |
| **Champs personnalisés** | Données additionnelles | Non |

## Limites par plan

| Plan | Limite de contacts |
|------|-------------------|
| Free | 300 contacts |
| Starter | Illimité |
| Pro | Illimité |

## Fonctionnalités

### Organisation

- **Groupes** : Regroupez vos contacts par thématique, source ou critère
- **Tags** : Ajoutez des étiquettes pour filtrer facilement
- **Champs personnalisés** : Créez vos propres attributs

### Segmentation

- Filtrez par groupe, tag ou valeur de champ
- Combinez plusieurs critères
- Exportez des segments

### Pipeline

Suivez l'avancement de vos prospects avec le pipeline CRM :
- Lead → Qualifié → Proposition → Négociation → Gagné/Perdu

### Import/Export

- Import CSV avec mapping automatique
- Export de tous vos contacts ou d'une sélection
- Détection des doublons

## Statuts des contacts

| Statut | Description |
|--------|-------------|
| **Actif** | Contact disponible pour les campagnes |
| **Désinscrit** | A cliqué sur le lien de désinscription |
| **Bounced** | Email invalide (retour en erreur) |
| **Répondu** | A répondu à une campagne |
| **Bloqué** | Bloqué manuellement |

## Bonnes pratiques

### Qualité des données

- Vérifiez les emails avant import
- Utilisez des services de validation (Hunter, Neverbounce)
- Nettoyez régulièrement les bounces

### Conformité RGPD

- Collectez le consentement quand nécessaire
- Respectez les demandes de désinscription
- Documentez vos sources de collecte

### Organisation

- Créez des groupes logiques dès le début
- Utilisez des conventions de nommage claires
- Archivez les contacts obsolètes
