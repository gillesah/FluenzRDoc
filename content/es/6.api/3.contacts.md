## Listar contactos

Recupera la lista paginada de contactos.

```
GET /contacts
```

### Parámetros de consulta

| Parámetro | Tipo | Descripción |
|-----------|------|-------------|
| `page` | integer | Número de página (predeterminado: 1) |
| `perPage` | integer | Elementos por página (predeterminado: 20, máx: 100) |
| `email` | string | Filtrar por correo (contiene) |
| `status` | string | Filtrar por estado |
| `groupId` | string | Filtrar por grupo |
| `tag` | string | Filtrar por etiqueta |
| `sort` | string | Campo de ordenación |
| `order` | string | asc o desc |

### Ejemplo

```bash
curl -X GET "https://api.fluenzr.co/v1/contacts?groupId=grp_xyz789&status=active&perPage=50" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### Respuesta

```json
{
  "success": true,
  "data": [
    {
      "id": "con_abc123",
      "email": "juan.perez@example.com",
      "firstName": "Juan",
      ...
    },
    ...
  ],
  "meta": {
    "page": 1,
    "perPage": 50,
    "total": 234,
    "totalPages": 5
  }
}
```

## Obtener un contacto

Recupera los detalles de un contacto específico.

```
GET /contacts/:id
```

### Ejemplo

```bash
curl -X GET "https://api.fluenzr.co/v1/contacts/con_abc123" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### Respuesta

```json
{
  "success": true,
  "data": {
    "id": "con_abc123",
    "email": "juan.perez@example.com",
    ...
  }
}
```

## Crear un contacto

Crea un nuevo contacto.

```
POST /contacts
```

### Cuerpo de la solicitud

| Campo | Tipo | Requerido | Descripción |
|-------|------|-----------|-------------|
| `email` | string | Sí | Dirección de correo (única) |
| `firstName` | string | | Nombre |
| `lastName` | string | | Apellido |
| `company` | string | | Empresa |
| `jobTitle` | string | | Puesto |
| `phone` | string | | Teléfono |
| `website` | string | | Sitio web |
| `groupIds` | array | | IDs de grupos |
| `tags` | array | | Lista de etiquetas |
| `customFields` | object | | Campos personalizados |
| `pipelineStageId` | string | | ID de etapa del pipeline |

### Ejemplo

```bash
curl -X POST "https://api.fluenzr.co/v1/contacts" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "maria.garcia@startup.io",
    "firstName": "María",
    "lastName": "García",
    "company": "StartupIO",
    "jobTitle": "CEO",
    "groupIds": ["grp_xyz789"],
    "tags": ["prospecto", "saas"],
    "customFields": {
      "sector": "SaaS",
      "tamano_empresa": "10-50"
    }
  }'
```

### Respuesta (201 Created)

```json
{
  "success": true,
  "data": {
    "id": "con_def456",
    "email": "maria.garcia@startup.io",
    ...
  }
}
```

### Posibles errores

| Código | Mensaje | Descripción |
|--------|---------|-------------|
| 400 | Email is required | Correo faltante |
| 400 | Invalid email format | Formato de correo inválido |
| 409 | Email already exists | Correo ya en uso |

## Actualizar un contacto

Actualiza un contacto existente.

```
PUT /contacts/:id
```

### Cuerpo de la solicitud

Los mismos campos que para la creación. Solo se actualizan los campos proporcionados.

### Ejemplo

```bash
curl -X PUT "https://api.fluenzr.co/v1/contacts/con_abc123" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "jobTitle": "VP Ventas",
    "customFields": {
      "tamano_empresa": "100-500"
    }
  }'
```

### Respuesta

```json
{
  "success": true,
  "data": {
    "id": "con_abc123",
    "jobTitle": "VP Ventas",
    ...
  }
}
```

## Eliminar un contacto

Elimina permanentemente un contacto.

```
DELETE /contacts/:id
```

### Ejemplo

```bash
curl -X DELETE "https://api.fluenzr.co/v1/contacts/con_abc123" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### Respuesta (200 OK)

```json
{
  "success": true,
  "data": {
    "id": "con_abc123",
    "deleted": true
  }
}
```

## Creación masiva

Crea múltiples contactos en una sola solicitud.

```
POST /contacts/bulk
```

### Cuerpo de la solicitud

```json
{
  "contacts": [
    {
      "email": "contacto1@example.com",
      "firstName": "Contacto",
      "lastName": "Uno"
    },
    {
      "email": "contacto2@example.com",
      "firstName": "Contacto",
      "lastName": "Dos"
    }
  ],
  "groupIds": ["grp_xyz789"],
  "onDuplicate": "skip"
}
```

| Parámetro | Descripción |
|-----------|-------------|
| `contacts` | Array de contactos (máx 1000) |
| `groupIds` | Grupos para asignar a todos |
| `onDuplicate` | Acción si existe duplicado: `skip`, `update` |

### Ejemplo

```bash
curl -X POST "https://api.fluenzr.co/v1/contacts/bulk" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "contacts": [
      {"email": "a@example.com", "firstName": "A"},
      {"email": "b@example.com", "firstName": "B"}
    ],
    "onDuplicate": "skip"
  }'
```

### Respuesta

```json
{
  "success": true,
  "data": {
    "created": 2,
    "updated": 0,
    "skipped": 0,
    "errors": []
  }
}
```

## Buscar contactos

Búsqueda avanzada de contactos.

```
POST /contacts/search
```

### Cuerpo de la solicitud

```json
{
  "query": "example.com",
  "filters": {
    "status": "active",
    "groups": ["grp_xyz789"],
    "customFields": {
      "sector": "Fintech"
    }
  },
  "sort": {
    "field": "createdAt",
    "order": "desc"
  }
}
```

### Ejemplo

```bash
curl -X POST "https://api.fluenzr.co/v1/contacts/search" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "filters": {
      "customFields": {
        "sector": "Fintech"
      }
    }
  }'
```
