# API de Campañas

Endpoints para crear, controlar y analizar tus campañas.

```json
{
  "id": "cmp_abc123",
```

## Listar campañas

Recupera la lista de campañas.

```
GET /campaigns
```

### Parámetros de consulta

| Parámetro | Tipo | Descripción |
|-----------|------|-------------|
| `page` | integer | Número de página |
| `perPage` | integer | Elementos por página |
| `status` | string | Filtrar por estado |
| `sort` | string | Campo de ordenación |
| `order` | string | asc o desc |

### Estados posibles

| Estado | Descripción |
|--------|-------------|
| `draft` | Borrador |
| `active` | En curso |
| `paused` | Pausada |
| `completed` | Completada |
| `archived` | Archivada |

### Ejemplo

```bash
curl -X GET "https://api.fluenzr.co/v1/campaigns?status=active" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### Respuesta

```json
{
  "success": true,
  "data": [
    {
      "id": "cmp_abc123",
      "name": "Prospección Q1 2025",
      "status": "active",
      ...
    }
  ],
  "meta": {
    "page": 1,
    "perPage": 20,
    "total": 15
  }
}
```

## Obtener una campaña

```
GET /campaigns/:id
```

### Ejemplo

```bash
curl -X GET "https://api.fluenzr.co/v1/campaigns/cmp_abc123" \
  -H "Authorization: Bearer flz_sk_xxx"
```

## Crear una campaña

```
POST /campaigns
```

### Cuerpo de la solicitud

| Campo | Tipo | Requerido | Descripción |
|-------|------|-----------|-------------|
| `name` | string | Sí | Nombre de la campaña |
| `description` | string | | Descripción |
| `emailAccountId` | string | Sí | ID de la cuenta de correo |
| `settings` | object | | Configuración de envío |
| `workflow` | object | | Estructura del flujo de trabajo |

### Estructura de settings

```json
{
  "sendDays": ["monday", "tuesday", "wednesday", "thursday", "friday"],
  "sendHoursStart": 9,
  "sendHoursEnd": 18,
  "timezone": "Europe/Madrid",
  "maxEmailsPerHour": 30
}
```

### Estructura de workflow

```json
{
  "nodes": [
    {
      "id": "node_1",
      "type": "start"
    },
    {
      "id": "node_2",
      "type": "email",
      "data": {
        "subject": "{{firstName}}, pregunta rápida",
        "body": "Hola {{firstName}},\n\nMe pongo en contacto...",
        "trackOpens": true,
        "trackClicks": true
      }
    },
    {
      "id": "node_3",
      "type": "delay",
      "data": {
        "days": 3,
        "businessDays": true
      }
    },
    {
      "id": "node_4",
      "type": "email",
      "data": {
        "subject": "Seguimiento de mi mensaje",
        "body": "{{firstName}},\n\nHago seguimiento..."
      }
    },
    {
      "id": "node_5",
      "type": "end"
    }
  ],
  "edges": [
    {"from": "node_1", "to": "node_2"},
    {"from": "node_2", "to": "node_3"},
    {"from": "node_3", "to": "node_4"},
    {"from": "node_4", "to": "node_5"}
  ]
}
```

### Ejemplo completo

```bash
curl -X POST "https://api.fluenzr.co/v1/campaigns" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Prospección Q1 2025",
    "emailAccountId": "acc_xyz789",
    "settings": {
      "sendDays": ["monday", "tuesday", "wednesday", "thursday", "friday"],
      "sendHoursStart": 9,
      "sendHoursEnd": 18,
      "timezone": "Europe/Madrid"
    },
    "workflow": {
      "nodes": [...],
      "edges": [...]
    }
  }'
```

## Actualizar una campaña

```
PUT /campaigns/:id
```

> **Advertencia**
Solo se pueden modificar campañas en estado `draft` o `paused`.

### Ejemplo

```bash
curl -X PUT "https://api.fluenzr.co/v1/campaigns/cmp_abc123" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Prospección Q1 2025 - v2",
    "settings": {
      "maxEmailsPerHour": 50
    }
  }'
```

## Controlar una campaña

### Iniciar

Lanza una campaña en borrador.

```
POST /campaigns/:id/start
```

```bash
curl -X POST "https://api.fluenzr.co/v1/campaigns/cmp_abc123/start" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### Pausar

Suspende una campaña activa.

```
POST /campaigns/:id/pause
```

### Reanudar

Reinicia una campaña pausada.

```
POST /campaigns/:id/resume
```

### Archivar

Archiva una campaña completada.

```
POST /campaigns/:id/archive
```

## Gestionar contactos de campaña

### Añadir contactos

```
POST /campaigns/:id/contacts
```

```json
{
  "contactIds": ["con_abc", "con_def"],
  // O
  "groupIds": ["grp_xyz"],
  // O
  "filters": {
    "status": "active",
    "customFields": {
      "sector": "Fintech"
    }
  }
}
```

### Ejemplo

```bash
curl -X POST "https://api.fluenzr.co/v1/campaigns/cmp_abc123/contacts" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "groupIds": ["grp_xyz789"]
  }'
```

### Respuesta

```json
{
  "success": true,
  "data": {
    "added": 150,
    "skipped": 5,
    "errors": []
  }
}
```

### Quitar contactos

```
DELETE /campaigns/:id/contacts
```

```json
{
  "contactIds": ["con_abc", "con_def"]
}
```

### Listar contactos

```
GET /campaigns/:id/contacts
```

| Parámetro | Descripción |
|-----------|-------------|
| `status` | Filtrar por estado (pending, in_progress, completed, replied, bounced) |
| `page` | Página |
| `perPage` | Elementos por página |
