# API REST

La API de FluenzR te permite integrar la plataforma con tus herramientas y automatizar tus flujos de trabajo.

> **Información**
La API REST solo está disponible en el plan **Pro**.

## Vista general

La API de FluenzR es una API REST que te permite:

- Gestionar tus contactos programáticamente
- Crear y controlar campañas
- Recuperar estadísticas
- Recibir webhooks sobre eventos

## URL base

```
https://api.fluenzr.co/v1
```

## Formato de respuestas

Todas las respuestas están en formato JSON:

```json
{
  "success": true,
  "data": { ... },
  "meta": {
    "page": 1,
    "perPage": 20,
    "total": 150
  }
}
```

### En caso de error

```json
{
  "success": false,
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Email is required",
    "details": { ... }
  }
}
```

## Códigos HTTP

| Código | Significado |
|--------|-------------|
| 200 | Éxito |
| 201 | Recurso creado |
| 400 | Solicitud inválida |
| 401 | No autenticado |
| 403 | No autorizado |
| 404 | Recurso no encontrado |
| 429 | Límite de tasa excedido |
| 500 | Error del servidor |

## Limitación de tasa

| Plan | Límite |
|------|--------|
| Pro | 1000 solicitudes/minuto |

Los encabezados de respuesta indican tu uso:

```
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 950
X-RateLimit-Reset: 1640000000
```

## Paginación

Los endpoints que listan recursos admiten paginación:

```
GET /contacts?page=2&perPage=50
```

| Parámetro | Descripción | Predeterminado | Máx |
|-----------|-------------|----------------|-----|
| `page` | Número de página | 1 | - |
| `perPage` | Elementos por página | 20 | 100 |

## Filtrado

Filtra resultados con parámetros de consulta:

```
GET /contacts?email=example.com&group=123
```

## Ordenación

Ordena los resultados:

```
GET /contacts?sort=createdAt&order=desc
```

## Endpoints disponibles

### Contactos

| Método | Endpoint | Descripción |
|--------|----------|-------------|
| GET | /contacts | Listar contactos |
| GET | /contacts/:id | Obtener un contacto |
| POST | /contacts | Crear un contacto |
| PUT | /contacts/:id | Actualizar un contacto |
| DELETE | /contacts/:id | Eliminar un contacto |

### Grupos

| Método | Endpoint | Descripción |
|--------|----------|-------------|
| GET | /groups | Listar grupos |
| GET | /groups/:id | Obtener un grupo |
| POST | /groups | Crear un grupo |
| PUT | /groups/:id | Actualizar un grupo |
| DELETE | /groups/:id | Eliminar un grupo |
| POST | /groups/:id/contacts | Añadir contactos |

### Campañas

| Método | Endpoint | Descripción |
|--------|----------|-------------|
| GET | /campaigns | Listar campañas |
| GET | /campaigns/:id | Obtener una campaña |
| POST | /campaigns | Crear una campaña |
| PUT | /campaigns/:id | Actualizar una campaña |
| POST | /campaigns/:id/start | Iniciar |
| POST | /campaigns/:id/pause | Pausar |
| POST | /campaigns/:id/contacts | Añadir contactos |
| GET | /campaigns/:id/stats | Estadísticas |

### Webhooks

| Método | Endpoint | Descripción |
|--------|----------|-------------|
| GET | /webhooks | Listar webhooks |
| POST | /webhooks | Crear un webhook |
| DELETE | /webhooks/:id | Eliminar |

## SDKs y bibliotecas

### JavaScript/TypeScript

```bash
npm install @fluenzr/sdk
```

```javascript
import { FluenzrClient } from '@fluenzr/sdk';

const client = new FluenzrClient({
  apiKey: 'your-api-key'
});

const contacts = await client.contacts.list();
```

### Python

```bash
pip install fluenzr
```

```python
from fluenzr import FluenzrClient

client = FluenzrClient(api_key='your-api-key')
contacts = client.contacts.list()
```

## Ejemplos de uso

### Sincronizar un CRM

```javascript
// Obtener nuevos contactos de tu CRM
const crmContacts = await myCRM.getNewContacts();

// Crearlos en FluenzR
for (const contact of crmContacts) {
  await fluenzr.contacts.create({
    email: contact.email,
    firstName: contact.first_name,
    lastName: contact.last_name,
    company: contact.company
  });
}
```

### Automatizar una campaña

```javascript
// Crear una campaña vía API
const campaign = await fluenzr.campaigns.create({
  name: 'Automatizada Q1 2025',
  emailAccountId: 'acc_123',
  // ...
});

// Añadir contactos
await fluenzr.campaigns.addContacts(campaign.id, {
  groupId: 'grp_456'
});

// Iniciar
await fluenzr.campaigns.start(campaign.id);
```

### Exportar estadísticas

```javascript
// Obtener estadísticas de todas las campañas
const campaigns = await fluenzr.campaigns.list();

for (const campaign of campaigns) {
  const stats = await fluenzr.campaigns.getStats(campaign.id);
  console.log(`${campaign.name}: ${stats.openRate}% aperturas, ${stats.replyRate}% respuestas`);
}
```
