# Campañas

Las campañas son el corazón de FluenzR. Crea secuencias de emails automatizadas para involucrar a tus prospectos de manera personalizada y efectiva.

## ¿Qué es una campaña?

Una campaña de FluenzR es una secuencia de emails automatizada que:

- Envía emails personalizados a tus contactos
- Respeta los retrasos entre cada email
- Se detiene automáticamente si el contacto responde
- Rastrea aperturas, clics y respuestas

## Componentes de una campaña

### Workflow visual

El editor de workflow te permite crear tu secuencia visualmente:

```
[Inicio]
    ↓
[Email 1] → "Primer contacto"
    ↓
[Retraso 3 días]
    ↓
[Email 2] → "Seguimiento"
    ↓
[Retraso 5 días]
    ↓
[Email 3] → "Último seguimiento"
    ↓
[Fin]
```

### Tipos de nodos

| Nodo | Descripción |
|------|-------------|
| **Email** | Envía un email personalizado |
| **Retraso** | Espera X días/horas antes del siguiente nodo |
| **Condición** | Bifurca según un criterio (Plan Pro) |
| **Fin** | Termina la secuencia |

## Estados de una campaña

| Estado | Descripción |
|-------|-------------|
| **Borrador** | En creación, no activa |
| **Activa** | Actualmente en ejecución |
| **Pausada** | Temporalmente suspendida |
| **Completada** | Todos los contactos han terminado la secuencia |
| **Archivada** | Campaña archivada |

## Ciclo de vida de un contacto en una campaña

1. **Añadido**: El contacto entra en la campaña
2. **En curso**: El contacto progresa por la secuencia
3. **Respondió**: El contacto respondió → Secuencia detenida
4. **Completado**: El contacto recibió todos los emails
5. **Dado de baja**: El contacto se dio de baja
6. **Error**: Problema de envío (rebote, etc.)

## Funcionalidades

### Personalización

- Variables dinámicas: `{{firstName}}`, `{{company}}`, etc.
- Campos personalizados en emails
- Firma personalizable

### Seguimiento

- Seguimiento de aperturas
- Seguimiento de clics
- Detección de respuestas
- Estadísticas en tiempo real

### Optimización (Starter+)

- A/B testing en asuntos
- A/B testing en contenido
- Analíticas avanzadas

### Automatización avanzada (Pro)

- Workflows condicionales
- Webhooks salientes
- Integración API

## Límites por plan

| Plan | Emails/día | A/B Testing | Workflows condicionales |
|------|------------|-------------|----------------------|
| Free | 50 | No | No |
| Starter | Ilimitados* | Sí | No |
| Pro | Ilimitados* | Sí | Sí |

*Según los límites de tu proveedor de email
