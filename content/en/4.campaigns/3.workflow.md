# Éditeur de workflow

L'éditeur de workflow FluenzR vous permet de créer visuellement vos séquences d'emails avec une interface drag-and-drop intuitive.

## Interface

### Zones principales

| Zone | Description |
|------|-------------|
| **Barre d'outils** | Nœuds disponibles à glisser |
| **Canvas** | Zone de travail pour votre workflow |
| **Panneau de configuration** | Paramètres du nœud sélectionné |

### Navigation

- **Zoom** : Molette de souris ou boutons +/-
- **Déplacement** : Clic gauche maintenu sur le canvas
- **Sélection** : Clic sur un nœud

## Types de nœuds

### Démarrage

Le nœud de démarrage est automatiquement présent. C'est le point d'entrée de votre séquence.

- Ne peut pas être supprimé
- Ne peut pas être dupliqué
- Un seul par workflow

### Email

Envoie un email personnalisé au contact.

**Configuration :**

| Champ | Description |
|-------|-------------|
| **Nom** | Identifiant du nœud (interne) |
| **Objet** | Ligne d'objet de l'email |
| **Contenu** | Corps de l'email |
| **Tracking** | Ouvertures et clics |

**Fonctionnalités :**
- Éditeur de texte riche
- Variables dynamiques
- Aperçu en temps réel

### Délai

Pause entre deux actions.

**Configuration :**

| Paramètre | Options |
|-----------|---------|
| **Durée** | 1-30 jours ou 1-24 heures |
| **Type** | Jours calendaires ou jours ouvrés |

**Exemples :**
- "3 jours" : Attend 72 heures
- "3 jours ouvrés" : Attend 3 jours (lun-ven), peut devenir 5 jours calendaires

### Condition (Plan Pro)

Crée une branche selon un critère.

**Critères disponibles :**

| Critère | Description |
|---------|-------------|
| Email ouvert | Le contact a ouvert le dernier email |
| Lien cliqué | Le contact a cliqué sur un lien |
| Email précédent | État de l'email précédent |
| Champ contact | Valeur d'un champ du contact |

**Structure :**
```
[Condition]
   ├─ Si vrai → [Email A]
   └─ Si faux → [Email B]
```

### Fin

Marque la fin de la séquence pour cette branche.

- Termine proprement le parcours du contact
- Peut avoir plusieurs nœuds Fin (branches différentes)

## Créer des connexions

### Connecter deux nœuds

1. Cliquez sur le point de sortie (bas) d'un nœud
2. Glissez vers le point d'entrée (haut) du nœud suivant
3. Relâchez pour créer la connexion

### Supprimer une connexion

1. Cliquez sur la connexion (la ligne)
2. Appuyez sur **Suppr** ou cliquez sur l'icône poubelle

## Éditer un nœud Email

### Ouvrir l'éditeur

Double-cliquez sur le nœud Email pour ouvrir le panneau d'édition.

### Objet de l'email

```
{{firstName}}, question rapide
```

Conseils :
- Maximum 50 caractères
- Utilisez des variables pour personnaliser
- Évitez les majuscules et points d'exclamation

### Contenu de l'email

L'éditeur offre :

| Fonction | Description |
|----------|-------------|
| **Gras/Italique** | Mise en forme basique |
| **Listes** | Puces et numérotées |
| **Liens** | Hyperliens cliquables |
| **Variables** | Menu d'insertion des variables |

### Options avancées

| Option | Description | Défaut |
|--------|-------------|--------|
| Tracking ouvertures | Suivre qui ouvre | Activé |
| Tracking clics | Suivre les clics | Activé |

## Bonnes pratiques de workflow

### Structure recommandée

Une séquence efficace :

```
[Démarrage]
    ↓
[Email 1] - Introduction
    ↓
[Délai 3 jours]
    ↓
[Email 2] - Valeur ajoutée
    ↓
[Délai 4 jours]
    ↓
[Email 3] - Dernière chance
    ↓
[Fin]
```

### Nombre d'emails optimal

| Type de campagne | Emails recommandés |
|------------------|-------------------|
| Prospection froide | 3-4 emails |
| Nurturing | 5-7 emails |
| Événement | 2-3 emails |
| Relance client | 2-3 emails |

### Délais entre emails

| Contexte | Délai recommandé |
|----------|------------------|
| Entre email 1 et 2 | 3-4 jours |
| Entre email 2 et 3 | 4-5 jours |
| Entre email 3 et 4 | 5-7 jours |

> **⚠️ Attention**
Des délais trop courts peuvent agacer vos prospects. Des délais trop longs font perdre le momentum.

## Workflows conditionnels (Plan Pro)

### Exemple : Relance selon ouverture

```
[Email 1]
    ↓
[Délai 3 jours]
    ↓
[Condition : Email 1 ouvert ?]
   ├─ Oui → [Email 2 : Relance personnalisée]
   └─ Non → [Email 2 : Nouvel angle]
```

### Exemple : Qualification selon clic

```
[Email avec 2 liens]
    ↓
[Délai 2 jours]
    ↓
[Condition : Clic sur lien A ?]
   ├─ Oui → [Email orienté solution A]
   └─ Non → [Condition : Clic sur lien B ?]
              ├─ Oui → [Email orienté solution B]
              └─ Non → [Email générique]
```

## Raccourcis clavier

| Raccourci | Action |
|-----------|--------|
| `Ctrl + Z` | Annuler |
| `Ctrl + Y` | Refaire |
| `Suppr` | Supprimer le nœud sélectionné |
| `Ctrl + D` | Dupliquer le nœud |
| `Ctrl + S` | Sauvegarder |

## Résolution de problèmes

### "Workflow invalide"

Vérifiez que :
- Tous les nœuds sont connectés
- Il n'y a pas de boucle
- Chaque branche se termine par un nœud Fin

### "Email sans contenu"

Chaque nœud Email doit avoir :
- Un objet non vide
- Un contenu non vide

### Nœud non connecté

Un nœud grisé ou avec un indicateur rouge n'est pas connecté. Reliez-le au workflow ou supprimez-le.
