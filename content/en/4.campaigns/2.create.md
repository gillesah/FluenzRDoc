# Créer une campagne

Ce guide vous accompagne dans la création de votre première campagne FluenzR.

## Étape 1 : Accéder à la création

1. Dans le menu latéral, cliquez sur **Campagnes**
2. Cliquez sur le bouton **+ Nouvelle campagne**

## Étape 2 : Configurer les paramètres de base

### Informations générales

| Champ | Description | Exemple |
|-------|-------------|---------|
| **Nom** | Identifiant interne de la campagne | "Prospection Q1 2025 - Fintech" |
| **Description** | Note explicative (optionnel) | "Séquence 3 emails pour leads LinkedIn" |

### Compte email

Sélectionnez le compte email qui enverra les messages :
- Si vous avez plusieurs comptes, la rotation automatique est activée (Starter+)
- Vérifiez que le compte est bien connecté et actif

### Paramètres d'envoi

| Paramètre | Description | Recommandation |
|-----------|-------------|----------------|
| **Jours d'envoi** | Jours de la semaine autorisés | Lun-Ven |
| **Heures d'envoi** | Plage horaire | 9h-18h |
| **Fuseau horaire** | Référence temporelle | Europe/Paris |
| **Emails/heure max** | Limite horaire | 20-30 |

> **ℹ️ Info**
Les paramètres de timing permettent d'envoyer vos emails aux moments les plus pertinents et d'éviter les filtres anti-spam.

## Étape 3 : Créer le workflow

L'éditeur de workflow s'ouvre automatiquement.

### Ajouter le premier email

1. Le nœud **Démarrage** est déjà présent
2. Cliquez sur **+ Email** dans la barre d'outils
3. Connectez le nœud Email au Démarrage

### Configurer l'email

Cliquez sur le nœud Email pour l'éditer :

#### Objet

```
{{firstName}}, une question rapide sur {{company}}
```

#### Contenu

```
Bonjour {{firstName}},

Je me permets de vous contacter car j'ai vu que {{company}}
développe son activité dans le secteur {{secteur}}.

Chez FluenzR, nous aidons les entreprises comme la vôtre à
automatiser leur prospection tout en gardant une approche
personnalisée.

Seriez-vous disponible pour un appel de 15 minutes cette semaine ?

Cordialement,
[Votre signature]
```

#### Options

| Option | Description |
|--------|-------------|
| **Tracking ouvertures** | Activer le suivi des ouvertures |
| **Tracking clics** | Activer le suivi des clics |

### Ajouter un délai

1. Cliquez sur **+ Délai**
2. Configurez la durée (ex: 3 jours)
3. Connectez après le premier email

### Ajouter des emails de relance

Répétez le processus pour créer une séquence :

```
Email 1 → Délai 3 jours → Email 2 → Délai 5 jours → Email 3
```

### Finaliser le workflow

Ajoutez un nœud **Fin** pour marquer la fin de la séquence.

## Étape 4 : Ajouter des contacts

### Méthode 1 : Depuis un groupe

1. Cliquez sur **Ajouter des contacts**
2. Sélectionnez l'onglet **Groupes**
3. Cochez le(s) groupe(s) souhaité(s)
4. Cliquez sur **Ajouter**

### Méthode 2 : Sélection individuelle

1. Cliquez sur **Ajouter des contacts**
2. Sélectionnez l'onglet **Contacts**
3. Utilisez les filtres si nécessaire
4. Cochez les contacts souhaités
5. Cliquez sur **Ajouter**

### Vérification

Avant de continuer, vérifiez :
- Le nombre de contacts ajoutés
- Pas de doublons avec d'autres campagnes actives
- Tous ont une adresse email valide

## Étape 5 : Prévisualiser

Avant de lancer, prévisualisez vos emails :

1. Cliquez sur **Aperçu**
2. Sélectionnez un contact exemple
3. Vérifiez que les variables sont bien remplacées
4. Vérifiez le rendu de l'email

> **⚠️ Attention**
Envoyez un email test à votre propre adresse pour vérifier le rendu dans votre boîte de réception.

## Étape 6 : Lancer la campagne

1. Cliquez sur **Démarrer la campagne**
2. Relisez le résumé :
   - Nombre de contacts
   - Nombre d'emails dans la séquence
   - Paramètres d'envoi
3. Cliquez sur **Confirmer**

## Après le lancement

### Suivi en temps réel

- Les premiers emails partent selon vos paramètres
- Consultez l'onglet **Statistiques** pour suivre les métriques
- Les réponses apparaissent dans l'**Inbox**

### Actions possibles

| Action | Description |
|--------|-------------|
| **Pause** | Suspend temporairement l'envoi |
| **Reprendre** | Relance une campagne en pause |
| **Ajouter des contacts** | Intègre de nouveaux contacts |
| **Archiver** | Archive la campagne terminée |

## Conseils pour une campagne efficace

### Objets d'email

- Personnalisez avec le prénom ou l'entreprise
- Évitez les mots spam (GRATUIT, URGENT, etc.)
- Restez court (< 50 caractères)
- Testez avec l'A/B testing

### Contenu

- Allez droit au but
- Une seule question/action par email
- Personnalisez au maximum
- Évitez les pièces jointes volumineuses

### Timing

- Relancez à intervalles raisonnables (3-5 jours)
- Limitez à 3-4 emails maximum par séquence
- Envoyez pendant les heures de bureau

### Signature

- Incluez vos coordonnées
- Ajoutez votre photo LinkedIn si pertinent
- Gardez une signature sobre et professionnelle
