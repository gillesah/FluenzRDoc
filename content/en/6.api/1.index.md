# REST API

The FluenzR API lets you integrate the platform with your tools and automate your workflows.

> **Info**
The REST API is only available on the **Pro** plan.

## Overview

The FluenzR API is a REST API that allows you to:

- Manage your contacts programmatically
- Create and control campaigns
- Retrieve statistics
- Receive webhooks on events

## Base URL

```
https://api.fluenzr.co/v1
```

## Response format

All responses are in JSON format:

```json
{
  "success": true,
  "data": { ... },
  "meta": {
    "page": 1,
    "perPage": 20,
    "total": 150
  }
}
```

### On error

```json
{
  "success": false,
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Email is required",
    "details": { ... }
  }
}
```

## HTTP codes

| Code | Meaning |
|------|---------|
| 200 | Success |
| 201 | Resource created |
| 400 | Invalid request |
| 401 | Not authenticated |
| 403 | Not authorized |
| 404 | Resource not found |
| 429 | Rate limit exceeded |
| 500 | Server error |

## Rate limiting

| Plan | Limit |
|------|-------|
| Pro | 1000 requests/minute |

Response headers indicate your usage:

```
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 950
X-RateLimit-Reset: 1640000000
```

## Pagination

Endpoints listing resources support pagination:

```
GET /contacts?page=2&perPage=50
```

| Parameter | Description | Default | Max |
|-----------|-------------|---------|-----|
| `page` | Page number | 1 | - |
| `perPage` | Items per page | 20 | 100 |

## Filtering

Filter results with query parameters:

```
GET /contacts?email=example.com&group=123
```

## Sorting

Sort results:

```
GET /contacts?sort=createdAt&order=desc
```

## Available endpoints

### Contacts

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /contacts | List contacts |
| GET | /contacts/:id | Get a contact |
| POST | /contacts | Create a contact |
| PUT | /contacts/:id | Update a contact |
| DELETE | /contacts/:id | Delete a contact |

### Groups

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /groups | List groups |
| GET | /groups/:id | Get a group |
| POST | /groups | Create a group |
| PUT | /groups/:id | Update a group |
| DELETE | /groups/:id | Delete a group |
| POST | /groups/:id/contacts | Add contacts |

### Campaigns

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /campaigns | List campaigns |
| GET | /campaigns/:id | Get a campaign |
| POST | /campaigns | Create a campaign |
| PUT | /campaigns/:id | Update a campaign |
| POST | /campaigns/:id/start | Start |
| POST | /campaigns/:id/pause | Pause |
| POST | /campaigns/:id/contacts | Add contacts |
| GET | /campaigns/:id/stats | Statistics |

### Webhooks

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /webhooks | List webhooks |
| POST | /webhooks | Create a webhook |
| DELETE | /webhooks/:id | Delete |

## SDKs and libraries

### JavaScript/TypeScript

```bash
npm install @fluenzr/sdk
```

```javascript
import { FluenzrClient } from '@fluenzr/sdk';

const client = new FluenzrClient({
  apiKey: 'your-api-key'
});

const contacts = await client.contacts.list();
```

### Python

```bash
pip install fluenzr
```

```python
from fluenzr import FluenzrClient

client = FluenzrClient(api_key='your-api-key')
contacts = client.contacts.list()
```

## Usage examples

### Sync a CRM

```javascript
// Get new contacts from your CRM
const crmContacts = await myCRM.getNewContacts();

// Create them in FluenzR
for (const contact of crmContacts) {
  await fluenzr.contacts.create({
    email: contact.email,
    firstName: contact.first_name,
    lastName: contact.last_name,
    company: contact.company
  });
}
```

### Automate a campaign

```javascript
// Create a campaign via API
const campaign = await fluenzr.campaigns.create({
  name: 'Automated Q1 2025',
  emailAccountId: 'acc_123',
  // ...
});

// Add contacts
await fluenzr.campaigns.addContacts(campaign.id, {
  groupId: 'grp_456'
});

// Start
await fluenzr.campaigns.start(campaign.id);
```

### Export statistics

```javascript
// Get stats for all campaigns
const campaigns = await fluenzr.campaigns.list();

for (const campaign of campaigns) {
  const stats = await fluenzr.campaigns.getStats(campaign.id);
  console.log(`${campaign.name}: ${stats.openRate}% opens, ${stats.replyRate}% replies`);
}
```
