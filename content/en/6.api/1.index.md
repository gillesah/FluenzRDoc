# API REST

L'API FluenzR vous permet d'intégrer la plateforme à vos outils et d'automatiser vos workflows.

> **ℹ️ Info**
L'API REST est disponible uniquement sur le plan **Pro**.

## Vue d'ensemble

L'API FluenzR est une API REST qui vous permet de :

- Gérer vos contacts programmatiquement
- Créer et contrôler des campagnes
- Récupérer des statistiques
- Recevoir des webhooks sur les événements

## URL de base

```
https://api.fluenzr.co/v1
```

## Format des réponses

Toutes les réponses sont au format JSON :

```json
{
  "success": true,
  "data": { ... },
  "meta": {
    "page": 1,
    "perPage": 20,
    "total": 150
  }
}
```

### En cas d'erreur

```json
{
  "success": false,
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Email is required",
    "details": { ... }
  }
}
```

## Codes HTTP

| Code | Signification |
|------|---------------|
| 200 | Succès |
| 201 | Ressource créée |
| 400 | Requête invalide |
| 401 | Non authentifié |
| 403 | Non autorisé |
| 404 | Ressource non trouvée |
| 429 | Rate limit dépassé |
| 500 | Erreur serveur |

## Rate limiting

| Plan | Limite |
|------|--------|
| Pro | 1000 requêtes/minute |

Les headers de réponse indiquent votre utilisation :

```
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 950
X-RateLimit-Reset: 1640000000
```

## Pagination

Les endpoints listant des ressources supportent la pagination :

```
GET /contacts?page=2&perPage=50
```

| Paramètre | Description | Défaut | Max |
|-----------|-------------|--------|-----|
| `page` | Numéro de page | 1 | - |
| `perPage` | Éléments par page | 20 | 100 |

## Filtrage

Filtrez les résultats avec des paramètres de requête :

```
GET /contacts?email=example.com&group=123
```

## Tri

Triez les résultats :

```
GET /contacts?sort=createdAt&order=desc
```

## Endpoints disponibles

### Contacts

| Méthode | Endpoint | Description |
|---------|----------|-------------|
| GET | /contacts | Liste des contacts |
| GET | /contacts/:id | Détail d'un contact |
| POST | /contacts | Créer un contact |
| PUT | /contacts/:id | Modifier un contact |
| DELETE | /contacts/:id | Supprimer un contact |

| PUT | /groups/:id | Modifier un groupe |
| DELETE | /groups/:id | Supprimer un groupe |

## SDKs et bibliothèques

### JavaScript/TypeScript

```bash
npm install @fluenzr/sdk
```

```javascript
import { FluenzrClient } from '@fluenzr/sdk';

const client = new FluenzrClient({
  apiKey: 'your-api-key'
});

const contacts = await client.contacts.list();
```

### Python

```bash
pip install fluenzr
```

```python
from fluenzr import FluenzrClient

client = FluenzrClient(api_key='your-api-key')
contacts = client.contacts.list()
```

## Exemples d'utilisation

### Synchroniser un CRM

```javascript
// Récupérer les nouveaux contacts depuis votre CRM
const crmContacts = await myCRM.getNewContacts();

// Les créer dans FluenzR
for (const contact of crmContacts) {
  await fluenzr.contacts.create({
    email: contact.email,
    firstName: contact.first_name,
    lastName: contact.last_name,
    company: contact.company
  });
}
```

### Automatiser une campagne

```javascript
// Créer une campagne via l'API
const campaign = await fluenzr.campaigns.create({
  name: 'Automated Q1 2025',
  emailAccountId: 'acc_123',
  // ...
});

// Ajouter des contacts
await fluenzr.campaigns.addContacts(campaign.id, {
  groupId: 'grp_456'
});

// Démarrer
await fluenzr.campaigns.start(campaign.id);
```

### Exporter des statistiques

```javascript
// Récupérer les stats de toutes les campagnes
const campaigns = await fluenzr.campaigns.list();

for (const campaign of campaigns) {
  const stats = await fluenzr.campaigns.getStats(campaign.id);
  console.log(`${campaign.name}: ${stats.openRate}% opens, ${stats.replyRate}% replies`);
}
```
