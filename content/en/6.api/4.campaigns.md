# Campaigns API

## List campaigns

Retrieves the list of campaigns.

```
GET /campaigns
```

### Query parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `page` | integer | Page number |
| `perPage` | integer | Items per page |
| `status` | string | Filter by status |
| `sort` | string | Sort field |
| `order` | string | asc or desc |

### Possible statuses

| Status | Description |
|--------|-------------|
| `draft` | Draft |
| `active` | Running |
| `paused` | Paused |
| `completed` | Completed |
| `archived` | Archived |

### Example

```bash
curl -X GET "https://api.fluenzr.co/v1/campaigns?status=active" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### Response

```json
{
  "success": true,
  "data": [
    {
      "id": "cmp_abc123",
      "name": "Q1 2025 Prospecting",
      "status": "active",
      ...
    }
  ],
  "meta": {
    "page": 1,
    "perPage": 20,
    "total": 15
  }
}
```


### Example

```bash

  -H "Authorization: Bearer flz_sk_xxx"
```

## Create a campaign

```
POST /campaigns
```

### Request body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | Yes | Campaign name |
| `description` | string | | Description |
| `emailAccountId` | string | Yes | Email account ID |
| `settings` | object | | Sending settings |
| `workflow` | object | | Workflow structure |

### Settings structure

```json
{
  "sendDays": ["monday", "tuesday", "wednesday", "thursday", "friday"],
  "sendHoursStart": 9,
  "sendHoursEnd": 18,
  "timezone": "Europe/Paris",
  "maxEmailsPerHour": 30
}
```

### Workflow structure

```json
{
  "nodes": [
    {
      "id": "node_1",
      "type": "start"
    },
    {
      "id": "node_2",
      "type": "email",
      "data": {
        "subject": "{{firstName}}, quick question",
        "body": "Hello {{firstName}},\n\nI'm reaching out...",
        "trackOpens": true,
        "trackClicks": true
      }
    },
    {
      "id": "node_3",
      "type": "delay",
      "data": {
        "days": 3,
        "businessDays": true
      }
    },
    {
      "id": "node_4",
      "type": "email",
      "data": {
        "subject": "Following up on my message",
        "body": "{{firstName}},\n\nI'm following up..."
      }
    },
    {
      "id": "node_5",
      "type": "end"
    }
  ],
  "edges": [
    {"from": "node_1", "to": "node_2"},
    {"from": "node_2", "to": "node_3"},
    {"from": "node_3", "to": "node_4"},
    {"from": "node_4", "to": "node_5"}
  ]
}
```

### Complete example

```bash
curl -X POST "https://api.fluenzr.co/v1/campaigns" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Q1 2025 Prospecting",
    "emailAccountId": "acc_xyz789",
    "settings": {
      "sendDays": ["monday", "tuesday", "wednesday", "thursday", "friday"],
      "sendHoursStart": 9,
      "sendHoursEnd": 18,
      "timezone": "Europe/Paris"
    },
    "workflow": {
      "nodes": [...],
      "edges": [...]
    }
  }'
```

## Update a campaign

```
PUT /campaigns/:id
```

> **Warning**
Only campaigns in `draft` or `paused` status can be modified.

### Example

```bash
curl -X PUT "https://api.fluenzr.co/v1/campaigns/cmp_abc123" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Q1 2025 Prospecting - v2",
    "settings": {
      "maxEmailsPerHour": 50
    }
  }'
```

## Control a campaign

### Start

Launches a draft campaign.

```
POST /campaigns/:id/start
```

```bash
curl -X POST "https://api.fluenzr.co/v1/campaigns/cmp_abc123/start" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### Pause

Suspends an active campaign.

```
POST /campaigns/:id/pause
```

### Resume

Restarts a paused campaign.

```
POST /campaigns/:id/resume
```

### Archive

Archives a completed campaign.

```
POST /campaigns/:id/archive
```

## Manage campaign contacts

### Add contacts

```
POST /campaigns/:id/contacts
```

```json
{
  "contactIds": ["con_abc", "con_def"],
  // OR
  "groupIds": ["grp_xyz"],
  // OR
  "filters": {
    "status": "active",
    "customFields": {
      "sector": "Fintech"
    }
  }
}
```

### Example

```bash
curl -X POST "https://api.fluenzr.co/v1/campaigns/cmp_abc123/contacts" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "groupIds": ["grp_xyz789"]
  }'
```

### Response

```json
{
  "success": true,
  "data": {
    "added": 150,
    "skipped": 5,
    "errors": []
  }
}
```

### Remove contacts

```
DELETE /campaigns/:id/contacts
```

```json
{
  "contactIds": ["con_abc", "con_def"]
}
```

### List contacts

```
GET /campaigns/:id/contacts
```

| Parameter | Description |
|-----------|-------------|
| `status` | Filter by status (pending, in_progress, completed, replied, bounced) |
| `page` | Page |
| `perPage` | Items per page |
