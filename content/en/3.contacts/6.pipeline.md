# CRM Pipeline

The FluenzR pipeline allows you to visually track your prospects' progress in your sales process.

## Overview

The pipeline is a Kanban view that represents the stages of your sales cycle. Each contact can be positioned in a pipeline stage.

### Default stages

| Stage | Description | Color |
|-------|-------------|-------|
| **Lead** | First contact, not yet qualified | Gray |
| **Qualified** | Interest confirmed, need identified | Blue |
| **Proposal** | Commercial proposal sent | Yellow |
| **Negotiation** | Discussion in progress | Orange |
| **Won** | Deal closed | Green |
| **Lost** | Opportunity lost | Red |

## Access the pipeline

1. In the sidebar, click on **Pipeline**
2. The Kanban view displays with all your positioned contacts

## Position a contact in the pipeline

### From the Pipeline view

1. Drag and drop the contact from one column to another
2. The change is saved automatically

### From the contact card

1. Open a contact's card
2. In the "Pipeline" section, select the stage from the dropdown menu
3. The change is saved automatically

### From the contacts list

1. In the contacts table, the "Pipeline" column displays the current stage
2. Click on the dropdown menu to change the stage directly

## Customize the pipeline

### Add a stage

1. In the Pipeline view, click on **+ Add a stage**
2. Define the name and color
3. Position the stage in the desired order

### Edit a stage

1. Click on the stage menu (⋮)
2. Select **Edit**
3. Change the name or color
4. Save

### Reorder stages

1. Drag and drop the stage header
2. Reposition it in the desired order

### Delete a stage

1. Click on the stage menu (⋮)
2. Select **Delete**
3. Choose where to move existing contacts
4. Confirm

> **Warning**
You cannot delete a stage that contains contacts without moving them first.

## Filter the pipeline

### By group

Display only contacts from a specific group:
1. Click on **Filters**
2. Select the group

### By period

Filter by the date of entry into the stage:
1. Click on **Filters**
2. Select a date range

### By owner

If you work as a team (Pro Plan):
1. Click on **Filters**
2. Select a team member

## Pipeline statistics

### Overview

At the top of the pipeline, view:
- Total number of contacts per stage
- Potential value (if "Deal value" field is filled)
- Conversion rate between stages

### Key metrics

| Metric | Description |
|--------|-------------|
| **Lead → Qualified conversion** | % of qualified leads |
| **Average time per stage** | Average duration before moving to the next stage |
| **Closing rate** | % of won deals |
| **Average value** | Average value of won deals |

## Integration with campaigns

### Possible automations

The pipeline integrates with your campaigns:

- **Trigger**: Contact added to a campaign → Moves to "Lead"
- **Response detected**: → Moves to "Qualified"
- **Manual action**: Send proposal → Moves to "Proposal"

### Interaction tracking

In the contact card, view the complete history:
- Emails sent via campaigns
- Opens and clicks
- Replies received
- Stage changes

## Best practices

### Define your stages

Adapt the stages to your actual process:

**B2B SaaS example:**
1. Lead
2. Demo scheduled
3. Demo completed
4. Proposal
5. Closing
6. Won / Lost

**Agency example:**
1. Prospect
2. First call
3. Brief sent
4. Quote
5. Negotiation
6. Signed / Lost

### Keep the pipeline up to date

- Move contacts as soon as they progress
- Regularly review "stuck" contacts
- Clean up lost opportunities

### Use notes

Add notes to each stage change:
- Reason for the move
- Next planned action
- Points of attention

## Team work (Pro Plan)

### Contact assignment

Assign contacts to team members:
1. Open the contact's card
2. Select the owner

### View by member

Filter the pipeline by member to see:
- Each person's opportunities
- Workload
- Individual performance

### Permissions

| Role | Pipeline permissions |
|------|---------------------|
| Admin | Everything (config + data) |
| Member | View and edit their contacts |
