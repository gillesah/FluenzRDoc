# Contact Management

Contacts are at the heart of your campaigns. FluenzR offers you powerful tools to import, organize, and segment them.

## Overview

A contact in FluenzR contains:

| Field | Description | Required |
|-------|-------------|----------|
| **Email** | Unique email address | Yes |
| **First name** | Contact's first name | No |
| **Last name** | Family name | No |
| **Company** | Company name | No |
| **Job title** | Position/title | No |
| **Phone** | Phone number | No |
| **Custom fields** | Additional data | No |

## Limits by plan

| Plan | Contact limit |
|------|---------------|
| Free | 300 contacts |
| Starter | Unlimited |
| Pro | Unlimited |

## Features

### Organization

- **Groups**: Group your contacts by theme, source, or criteria
- **Tags**: Add labels for easy filtering
- **Custom fields**: Create your own attributes

### Segmentation

- Filter by group, tag, or field value
- Combine multiple criteria
- Export segments

### Pipeline

Track your prospect progress with the CRM pipeline:
- Lead → Qualified → Proposal → Negotiation → Won/Lost

### Import/Export

- CSV import with automatic mapping
- Export all your contacts or a selection
- Duplicate detection

## Contact statuses

| Status | Description |
|--------|-------------|
| **Active** | Contact available for campaigns |
| **Unsubscribed** | Clicked on the unsubscribe link |
| **Bounced** | Invalid email (bounce) |
| **Replied** | Responded to a campaign |
| **Blocked** | Manually blocked |

## Best practices

### Data quality

- Verify emails before import
- Use validation services (Hunter, Neverbounce)
- Regularly clean up bounces

### GDPR compliance

- Collect consent when necessary
- Respect unsubscribe requests
- Document your collection sources

### Organization

- Create logical groups from the start
- Use clear naming conventions
- Archive obsolete contacts
