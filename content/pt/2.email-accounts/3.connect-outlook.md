# Conectar Outlook

O FluenzR suporta contas pessoais Outlook e contas profissionais Microsoft 365 via OAuth 2.0.

## Pré-requisitos

- Uma conta Outlook.com, Hotmail ou Microsoft 365
- Acesso à página de contas de email no FluenzR

## Passos de conexão

### 1. Acessar a configuração

1. Faça login na sua conta FluenzR
2. No menu lateral, clique em **Contas de Email**
3. Clique no botão **+ Microsoft**

### 2. Autorizar acesso

1. Uma janela da Microsoft abre
2. Faça login na sua conta Microsoft
3. Revise as permissões solicitadas

### 3. Conceder permissões

O FluenzR solicita acesso para:
- **Mail.Send**: Enviar emails
- **Mail.Read**: Ler emails (detecção de respostas)
- **User.Read**: Acessar seu perfil básico

Clique em **Aceitar** para confirmar.

### 4. Verificação

Após a autorização, sua conta Outlook aparece no FluenzR com status **Conectado**.

## Limites do Outlook

| Tipo de conta | Limite diário |
|---------------|---------------|
| Outlook.com / Hotmail | ~300 emails/dia |
| Microsoft 365 Business | ~10.000 emails/dia |

> **Info**
Contas profissionais Microsoft 365 têm limites muito mais altos, tornando-as ideais para prospecção.

## Microsoft 365 vs Outlook.com

### Outlook.com (pessoal)
- Grátis
- Limites mais restritivos
- Adequado para pequenos volumes

### Microsoft 365 (profissional)
- Pago (a partir de €5,60/mês)
- Domínio personalizado possível
- Limites altos
- Administração centralizada
- Melhor imagem profissional

## Configuração para Microsoft 365

Se você usa Microsoft 365 com um domínio personalizado, certifique-se de que:

1. **SPF** está configurado para seu domínio
2. **DKIM** está habilitado no admin do Microsoft 365
3. **DMARC** está configurado (recomendado)

Essas configurações DNS melhoram a entregabilidade dos seus emails.

## Solução de problemas

### "Acesso negado pelo administrador"
Para contas Microsoft 365, o administrador da sua organização pode bloquear aplicativos de terceiros:
1. Entre em contato com seu administrador de TI
2. Solicite a aprovação do FluenzR no Azure AD
3. Ou use uma conta pessoal enquanto isso

### "Falha na autenticação"
- Verifique se a conta não tem autenticação de dois fatores bloqueante
- Tente fazer logout/login novamente na Microsoft
- Use um navegador privado

### "Token inválido"
- No FluenzR, exclua a conta
- Reconecte-a desde o início
- Se o problema persistir, revogue o acesso no lado da Microsoft e reconecte

## Revogar acesso

Para desconectar o FluenzR da sua conta Microsoft:

1. Acesse [account.microsoft.com/privacy/app-access](https://account.microsoft.com/privacy/app-access)
2. Encontre "FluenzR" na lista
3. Clique em **Remover essas permissões**

## Comparação Gmail vs Outlook

| Critério | Gmail | Outlook |
|----------|-------|---------|
| Limite pessoal | 500/dia | 300/dia |
| Limite profissional | 2000/dia | 10000/dia |
| Configuração | Simples | Simples |
| Entregabilidade | Excelente | Muito boa |
| Preço profissional | €5,75/mês | €5,60/mês |

> **Sucesso**
Para máxima flexibilidade, combine contas Gmail e Outlook no seu plano Starter ou Pro.
