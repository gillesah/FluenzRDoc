# API REST

A API do FluenzR permite integrar a plataforma às suas ferramentas e automatizar seus fluxos de trabalho.

> **Info**
A API REST está disponível apenas no plano **Pro**.

## Visão Geral

A API do FluenzR é uma API REST que permite:

- Gerenciar seus contatos programaticamente
- Criar e controlar campanhas
- Recuperar estatísticas
- Receber webhooks sobre eventos

## URL Base

```
https://api.fluenzr.co/v1
```

## Formato das respostas

Todas as respostas estão no formato JSON:

```json
{
  "success": true,
  "data": { ... },
  "meta": {
    "page": 1,
    "perPage": 20,
    "total": 150
  }
}
```

### Em caso de erro

```json
{
  "success": false,
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Email is required",
    "details": { ... }
  }
}
```

## Códigos HTTP

| Código | Significado |
|--------|-------------|
| 200 | Sucesso |
| 201 | Recurso criado |
| 400 | Solicitação inválida |
| 401 | Não autenticado |
| 403 | Não autorizado |
| 404 | Recurso não encontrado |
| 429 | Limite de taxa excedido |
| 500 | Erro do servidor |

## Limitação de taxa

| Plano | Limite |
|-------|--------|
| Pro | 1000 requisições/minuto |

Os cabeçalhos de resposta indicam seu uso:

```
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 950
X-RateLimit-Reset: 1640000000
```

## Paginação

Endpoints que listam recursos suportam paginação:

```
GET /contacts?page=2&perPage=50
```

| Parâmetro | Descrição | Padrão | Máx |
|-----------|-----------|--------|-----|
| `page` | Número da página | 1 | - |
| `perPage` | Itens por página | 20 | 100 |

## Filtragem

Filtre resultados com parâmetros de consulta:

```
GET /contacts?email=example.com&group=123
```

## Ordenação

Ordene os resultados:

```
GET /contacts?sort=createdAt&order=desc
```

## Endpoints disponíveis

### Contatos

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| GET | /contacts | Listar contatos |
| GET | /contacts/:id | Obter um contato |
| POST | /contacts | Criar um contato |
| PUT | /contacts/:id | Atualizar um contato |
| DELETE | /contacts/:id | Excluir um contato |

### Grupos

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| GET | /groups | Listar grupos |
| GET | /groups/:id | Obter um grupo |
| POST | /groups | Criar um grupo |
| PUT | /groups/:id | Atualizar um grupo |
| DELETE | /groups/:id | Excluir um grupo |
| POST | /groups/:id/contacts | Adicionar contatos |

### Campanhas

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| GET | /campaigns | Listar campanhas |
| GET | /campaigns/:id | Obter uma campanha |
| POST | /campaigns | Criar uma campanha |
| PUT | /campaigns/:id | Atualizar uma campanha |
| POST | /campaigns/:id/start | Iniciar |
| POST | /campaigns/:id/pause | Pausar |
| POST | /campaigns/:id/contacts | Adicionar contatos |
| GET | /campaigns/:id/stats | Estatísticas |

### Webhooks

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| GET | /webhooks | Listar webhooks |
| POST | /webhooks | Criar um webhook |
| DELETE | /webhooks/:id | Excluir |

## SDKs e bibliotecas

### JavaScript/TypeScript

```bash
npm install @fluenzr/sdk
```

```javascript
import { FluenzrClient } from '@fluenzr/sdk';

const client = new FluenzrClient({
  apiKey: 'your-api-key'
});

const contacts = await client.contacts.list();
```

### Python

```bash
pip install fluenzr
```

```python
from fluenzr import FluenzrClient

client = FluenzrClient(api_key='your-api-key')
contacts = client.contacts.list()
```

## Exemplos de uso

### Sincronizar um CRM

```javascript
// Obter novos contatos do seu CRM
const crmContacts = await myCRM.getNewContacts();

// Criá-los no FluenzR
for (const contact of crmContacts) {
  await fluenzr.contacts.create({
    email: contact.email,
    firstName: contact.first_name,
    lastName: contact.last_name,
    company: contact.company
  });
}
```

### Automatizar uma campanha

```javascript
// Criar uma campanha via API
const campaign = await fluenzr.campaigns.create({
  name: 'Automated Q1 2025',
  emailAccountId: 'acc_123',
  // ...
});

// Adicionar contatos
await fluenzr.campaigns.addContacts(campaign.id, {
  groupId: 'grp_456'
});

// Iniciar
await fluenzr.campaigns.start(campaign.id);
```

### Exportar estatísticas

```javascript
// Obter estatísticas de todas as campanhas
const campaigns = await fluenzr.campaigns.list();

for (const campaign of campaigns) {
  const stats = await fluenzr.campaigns.getStats(campaign.id);
  console.log(`${campaign.name}: ${stats.openRate}% aberturas, ${stats.replyRate}% respostas`);
}
```
