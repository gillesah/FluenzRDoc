## Listar contatos

Recupera a lista paginada de contatos.

```
GET /contacts
```

### Parâmetros de consulta

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `page` | integer | Número da página (padrão: 1) |
| `perPage` | integer | Itens por página (padrão: 20, máx: 100) |
| `email` | string | Filtrar por email (contém) |
| `status` | string | Filtrar por status |
| `groupId` | string | Filtrar por grupo |
| `tag` | string | Filtrar por tag |
| `sort` | string | Campo de ordenação |
| `order` | string | asc ou desc |

### Exemplo

```bash
curl -X GET "https://api.fluenzr.co/v1/contacts?groupId=grp_xyz789&status=active&perPage=50" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### Resposta

```json
{
  "success": true,
  "data": [
    {
      "id": "con_abc123",
      "email": "joao.silva@example.com",
      "firstName": "João",
      ...
    },
    ...
  ],
  "meta": {
    "page": 1,
    "perPage": 50,
    "total": 234,
    "totalPages": 5
  }
}
```

## Obter um contato

Recupera detalhes de um contato específico.

```
GET /contacts/:id
```

### Exemplo

```bash
curl -X GET "https://api.fluenzr.co/v1/contacts/con_abc123" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### Resposta

```json
{
  "success": true,
  "data": {
    "id": "con_abc123",
    "email": "joao.silva@example.com",
    ...
  }
}
```

## Criar um contato

Cria um novo contato.

```
POST /contacts
```

### Corpo da solicitação

| Campo | Tipo | Obrigatório | Descrição |
|-------|------|-------------|-----------|
| `email` | string | Sim | Endereço de email (único) |
| `firstName` | string | | Primeiro nome |
| `lastName` | string | | Sobrenome |
| `company` | string | | Empresa |
| `jobTitle` | string | | Cargo |
| `phone` | string | | Telefone |
| `website` | string | | Site |
| `groupIds` | array | | IDs dos grupos |
| `tags` | array | | Lista de tags |
| `customFields` | object | | Campos personalizados |
| `pipelineStageId` | string | | ID da etapa do pipeline |

### Exemplo

```bash
curl -X POST "https://api.fluenzr.co/v1/contacts" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "maria.santos@startup.io",
    "firstName": "Maria",
    "lastName": "Santos",
    "company": "StartupIO",
    "jobTitle": "CEO",
    "groupIds": ["grp_xyz789"],
    "tags": ["prospecto", "saas"],
    "customFields": {
      "setor": "SaaS",
      "tamanho_empresa": "10-50"
    }
  }'
```

### Resposta (201 Created)

```json
{
  "success": true,
  "data": {
    "id": "con_def456",
    "email": "maria.santos@startup.io",
    ...
  }
}
```

### Erros possíveis

| Código | Mensagem | Descrição |
|--------|----------|-----------|
| 400 | Email is required | Email ausente |
| 400 | Invalid email format | Formato de email inválido |
| 409 | Email already exists | Email já em uso |

## Atualizar um contato

Atualiza um contato existente.

```
PUT /contacts/:id
```

### Corpo da solicitação

Mesmos campos da criação. Apenas os campos fornecidos são atualizados.

### Exemplo

```bash
curl -X PUT "https://api.fluenzr.co/v1/contacts/con_abc123" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "jobTitle": "VP de Vendas",
    "customFields": {
      "tamanho_empresa": "100-500"
    }
  }'
```

### Resposta

```json
{
  "success": true,
  "data": {
    "id": "con_abc123",
    "jobTitle": "VP de Vendas",
    ...
  }
}
```

## Excluir um contato

Exclui permanentemente um contato.

```
DELETE /contacts/:id
```

### Exemplo

```bash
curl -X DELETE "https://api.fluenzr.co/v1/contacts/con_abc123" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### Resposta (200 OK)

```json
{
  "success": true,
  "data": {
    "id": "con_abc123",
    "deleted": true
  }
}
```

## Criar em massa

Cria vários contatos em uma única solicitação.

```
POST /contacts/bulk
```

### Corpo da solicitação

```json
{
  "contacts": [
    {
      "email": "contato1@example.com",
      "firstName": "Contato",
      "lastName": "Um"
    },
    {
      "email": "contato2@example.com",
      "firstName": "Contato",
      "lastName": "Dois"
    }
  ],
  "groupIds": ["grp_xyz789"],
  "onDuplicate": "skip"
}
```

| Parâmetro | Descrição |
|-----------|-----------|
| `contacts` | Array de contatos (máx 1000) |
| `groupIds` | Grupos a atribuir a todos |
| `onDuplicate` | Ação em caso de duplicata: `skip`, `update` |

### Exemplo

```bash
curl -X POST "https://api.fluenzr.co/v1/contacts/bulk" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "contacts": [
      {"email": "a@example.com", "firstName": "A"},
      {"email": "b@example.com", "firstName": "B"}
    ],
    "onDuplicate": "skip"
  }'
```

### Resposta

```json
{
  "success": true,
  "data": {
    "created": 2,
    "updated": 0,
    "skipped": 0,
    "errors": []
  }
}
```

## Pesquisar contatos

Pesquisa avançada de contatos.

```
POST /contacts/search
```

### Corpo da solicitação

```json
{
  "query": "example.com",
  "filters": {
    "status": "active",
    "groups": ["grp_xyz789"],
    "customFields": {
      "setor": "Fintech"
    }
  },
  "sort": {
    "field": "createdAt",
    "order": "desc"
  }
}
```

### Exemplo

```bash
curl -X POST "https://api.fluenzr.co/v1/contacts/search" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "filters": {
      "customFields": {
        "setor": "Fintech"
      }
    }
  }'
```

## Gerenciar grupos de contatos

```
POST /contacts/:id/groups
```

```json
{
  "groupIds": ["grp_abc", "grp_def"]
}
```

### Remover de grupos

```
DELETE /contacts/:id/groups
```

```json
{
  "groupIds": ["grp_abc"]
}
```
