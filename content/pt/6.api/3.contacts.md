## Liste des contacts

Récupère la liste paginée des contacts.

```
GET /contacts
```

### Paramètres de requête

| Paramètre | Type | Description |
|-----------|------|-------------|
| `page` | integer | Numéro de page (défaut: 1) |
| `perPage` | integer | Éléments par page (défaut: 20, max: 100) |
| `email` | string | Filtrer par email (contient) |
| `status` | string | Filtrer par statut |
| `groupId` | string | Filtrer par groupe |
| `tag` | string | Filtrer par tag |
| `sort` | string | Champ de tri |
| `order` | string | asc ou desc |

### Exemple

```bash
curl -X GET "https://api.fluenzr.co/v1/contacts?groupId=grp_xyz789&status=active&perPage=50" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### Réponse

```json
{
  "success": true,
  "data": [
    {
      "id": "con_abc123",
      "email": "jean.dupont@example.com",
      "firstName": "Jean",
      ...
    },
    ...
  ],
  "meta": {
    "page": 1,
    "perPage": 50,
    "total": 234,
    "totalPages": 5
  }
}
```

## Récupérer un contact

Récupère les détails d'un contact spécifique.

```
GET /contacts/:id
```

### Exemple

```bash
curl -X GET "https://api.fluenzr.co/v1/contacts/con_abc123" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### Réponse

```json
{
  "success": true,
  "data": {
    "id": "con_abc123",
    "email": "jean.dupont@example.com",
    ...
  }
}
```

## Créer un contact

Crée un nouveau contact.

```
POST /contacts
```

### Corps de la requête

| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| `email` | string | ✅ | Adresse email (unique) |
| `firstName` | string | | Prénom |
| `lastName` | string | | Nom |
| `company` | string | | Entreprise |
| `jobTitle` | string | | Poste |
| `phone` | string | | Téléphone |
| `website` | string | | Site web |
| `groupIds` | array | | IDs des groupes |
| `tags` | array | | Liste de tags |
| `customFields` | object | | Champs personnalisés |
| `pipelineStageId` | string | | ID de l'étape pipeline |

### Exemple

```bash
curl -X POST "https://api.fluenzr.co/v1/contacts" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "marie.martin@startup.io",
    "firstName": "Marie",
    "lastName": "Martin",
    "company": "StartupIO",
    "jobTitle": "CEO",
    "groupIds": ["grp_xyz789"],
    "tags": ["prospect", "saas"],
    "customFields": {
      "secteur": "SaaS",
      "taille_entreprise": "10-50"
    }
  }'
```

### Réponse (201 Created)

```json
{
  "success": true,
  "data": {
    "id": "con_def456",
    "email": "marie.martin@startup.io",
    ...
  }
}
```

### Erreurs possibles

| Code | Message | Description |
|------|---------|-------------|
| 400 | Email is required | Email manquant |
| 400 | Invalid email format | Format email invalide |
| 409 | Email already exists | Email déjà utilisé |

## Modifier un contact

Met à jour un contact existant.

```
PUT /contacts/:id
```

### Corps de la requête

Mêmes champs que la création. Seuls les champs fournis sont mis à jour.

### Exemple

```bash
curl -X PUT "https://api.fluenzr.co/v1/contacts/con_abc123" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "jobTitle": "VP Sales",
    "customFields": {
      "taille_entreprise": "100-500"
    }
  }'
```

### Réponse

```json
{
  "success": true,
  "data": {
    "id": "con_abc123",
    "jobTitle": "VP Sales",
    ...
  }
}
```

## Supprimer un contact

Supprime définitivement un contact.

```
DELETE /contacts/:id
```

### Exemple

```bash
curl -X DELETE "https://api.fluenzr.co/v1/contacts/con_abc123" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### Réponse (200 OK)

```json
{
  "success": true,
  "data": {
    "id": "con_abc123",
    "deleted": true
  }
}
```

## Créer en masse

Crée plusieurs contacts en une seule requête.

```
POST /contacts/bulk
```

### Corps de la requête

```json
{
  "contacts": [
    {
      "email": "contact1@example.com",
      "firstName": "Contact",
      "lastName": "Un"
    },
    {
      "email": "contact2@example.com",
      "firstName": "Contact",
      "lastName": "Deux"
    }
  ],
  "groupIds": ["grp_xyz789"],
  "onDuplicate": "skip"
}
```

| Paramètre | Description |
|-----------|-------------|
| `contacts` | Array de contacts (max 1000) |
| `groupIds` | Groupes à assigner à tous |
| `onDuplicate` | Action si doublon : `skip`, `update` |

### Exemple

```bash
curl -X POST "https://api.fluenzr.co/v1/contacts/bulk" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "contacts": [
      {"email": "a@example.com", "firstName": "A"},
      {"email": "b@example.com", "firstName": "B"}
    ],
    "onDuplicate": "skip"
  }'
```

### Réponse

```json
{
  "success": true,
  "data": {
    "created": 2,
    "updated": 0,
    "skipped": 0,
    "errors": []
  }
}
```

## Rechercher des contacts

Recherche avancée dans les contacts.

```
POST /contacts/search
```

### Corps de la requête

```json
{
  "query": "example.com",
  "filters": {
    "status": "active",
    "groups": ["grp_xyz789"],
    "customFields": {
      "secteur": "Fintech"
    }
  },
  "sort": {
    "field": "createdAt",
    "order": "desc"
  }
}
```

### Exemple

```bash
curl -X POST "https://api.fluenzr.co/v1/contacts/search" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "filters": {
      "customFields": {
        "secteur": "Fintech"
      }
    }
  }'
```
