## Listar campanhas

Recupera a lista de campanhas.

```
GET /campaigns
```

### Parâmetros de consulta

| Parâmetro | Tipo | Descrição |
|-----------|------|-----------|
| `page` | integer | Número da página |
| `perPage` | integer | Itens por página |
| `status` | string | Filtrar por status |
| `sort` | string | Campo de ordenação |
| `order` | string | asc ou desc |

### Status possíveis

| Status | Descrição |
|--------|-----------|
| `draft` | Rascunho |
| `active` | Em andamento |
| `paused` | Pausada |
| `completed` | Concluída |
| `archived` | Arquivada |

### Exemplo

```bash
curl -X GET "https://api.fluenzr.co/v1/campaigns?status=active" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### Resposta

```json
{
  "success": true,
  "data": [
    {
      "id": "cmp_abc123",
      "name": "Prospecção Q1 2025",
      "status": "active",
      ...
    }
  ],
  "meta": {
    "page": 1,
    "perPage": 20,
    "total": 15
  }
}
```

## Obter uma campanha

```
GET /campaigns/:id
```

### Exemplo

```bash
curl -X GET "https://api.fluenzr.co/v1/campaigns/cmp_abc123" \
  -H "Authorization: Bearer flz_sk_xxx"
```

## Criar uma campanha

```
POST /campaigns
```

### Corpo da solicitação

| Campo | Tipo | Obrigatório | Descrição |
|-------|------|-------------|-----------|
| `name` | string | Sim | Nome da campanha |
| `description` | string | | Descrição |
| `emailAccountId` | string | Sim | ID da conta de email |
| `settings` | object | | Configurações de envio |
| `workflow` | object | | Estrutura do fluxo de trabalho |

### Estrutura de settings

```json
{
  "sendDays": ["monday", "tuesday", "wednesday", "thursday", "friday"],
  "sendHoursStart": 9,
  "sendHoursEnd": 18,
  "timezone": "America/Sao_Paulo",
  "maxEmailsPerHour": 30
}
```

### Estrutura de workflow

```json
{
  "nodes": [
    {
      "id": "node_1",
      "type": "start"
    },
    {
      "id": "node_2",
      "type": "email",
      "data": {
        "subject": "{{firstName}}, pergunta rápida",
        "body": "Olá {{firstName}},\n\nEstou entrando em contato...",
        "trackOpens": true,
        "trackClicks": true
      }
    },
    {
      "id": "node_3",
      "type": "delay",
      "data": {
        "days": 3,
        "businessDays": true
      }
    },
    {
      "id": "node_4",
      "type": "email",
      "data": {
        "subject": "Acompanhamento da minha mensagem",
        "body": "{{firstName}},\n\nRetornando sobre..."
      }
    },
    {
      "id": "node_5",
      "type": "end"
    }
  ],
  "edges": [
    {"from": "node_1", "to": "node_2"},
    {"from": "node_2", "to": "node_3"},
    {"from": "node_3", "to": "node_4"},
    {"from": "node_4", "to": "node_5"}
  ]
}
```

### Exemplo completo

```bash
curl -X POST "https://api.fluenzr.co/v1/campaigns" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Prospecção Q1 2025",
    "emailAccountId": "acc_xyz789",
    "settings": {
      "sendDays": ["monday", "tuesday", "wednesday", "thursday", "friday"],
      "sendHoursStart": 9,
      "sendHoursEnd": 18,
      "timezone": "America/Sao_Paulo"
    },
    "workflow": {
      "nodes": [...],
      "edges": [...]
    }
  }'
```

## Atualizar uma campanha

```
PUT /campaigns/:id
```

> **Atenção**
Apenas campanhas com status `draft` ou `paused` podem ser modificadas.

### Exemplo

```bash
curl -X PUT "https://api.fluenzr.co/v1/campaigns/cmp_abc123" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Prospecção Q1 2025 - v2",
    "settings": {
      "maxEmailsPerHour": 50
    }
  }'
```

## Controlar uma campanha

### Iniciar

Inicia uma campanha em rascunho.

```
POST /campaigns/:id/start
```

```bash
curl -X POST "https://api.fluenzr.co/v1/campaigns/cmp_abc123/start" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### Pausar

Suspende uma campanha ativa.

```
POST /campaigns/:id/pause
```

### Retomar

Reinicia uma campanha pausada.

```
POST /campaigns/:id/resume
```

### Arquivar

Arquiva uma campanha concluída.

```
POST /campaigns/:id/archive
```

## Gerenciar contatos de uma campanha

### Adicionar contatos

```
POST /campaigns/:id/contacts
```

```json
{
  "contactIds": ["con_abc", "con_def"],
  // OU
  "groupIds": ["grp_xyz"],
  // OU
  "filters": {
    "status": "active",
    "customFields": {
      "setor": "Fintech"
    }
  }
}
```

### Exemplo

```bash
curl -X POST "https://api.fluenzr.co/v1/campaigns/cmp_abc123/contacts" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "groupIds": ["grp_xyz789"]
  }'
```

### Resposta

```json
{
  "success": true,
  "data": {
    "added": 150,
    "skipped": 5,
    "errors": []
  }
}
```

### Remover contatos

```
DELETE /campaigns/:id/contacts
```

```json
{
  "contactIds": ["con_abc", "con_def"]
}
```

### Listar contatos

```
GET /campaigns/:id/contacts
```

| Parâmetro | Descrição |
|-----------|-----------|
| `status` | Filtrar por status (pending, in_progress, completed, replied, bounced) |
| `page` | Página |
| `perPage` | Itens por página |
