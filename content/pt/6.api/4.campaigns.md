## Liste des campagnes

Récupère la liste des campagnes.

```
GET /campaigns
```

### Paramètres de requête

| Paramètre | Type | Description |
|-----------|------|-------------|
| `page` | integer | Numéro de page |
| `perPage` | integer | Éléments par page |
| `status` | string | Filtrer par statut |
| `sort` | string | Champ de tri |
| `order` | string | asc ou desc |

### Statuts possibles

| Statut | Description |
|--------|-------------|
| `draft` | Brouillon |
| `active` | En cours |
| `paused` | En pause |
| `completed` | Terminée |
| `archived` | Archivée |

### Exemple

```bash
curl -X GET "https://api.fluenzr.co/v1/campaigns?status=active" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### Réponse

```json
{
  "success": true,
  "data": [
    {
      "id": "cmp_abc123",
      "name": "Prospection Q1 2025",
      "status": "active",
      ...
    }
  ],
  "meta": {
    "page": 1,
    "perPage": 20,
    "total": 15
  }
}
```

## Récupérer une campagne

```
GET /campaigns/:id
```

### Exemple

```bash
curl -X GET "https://api.fluenzr.co/v1/campaigns/cmp_abc123" \
  -H "Authorization: Bearer flz_sk_xxx"
```

## Créer une campagne

```
POST /campaigns
```

### Corps de la requête

| Champ | Type | Requis | Description |
|-------|------|--------|-------------|
| `name` | string | ✅ | Nom de la campagne |
| `description` | string | | Description |
| `emailAccountId` | string | ✅ | ID du compte email |
| `settings` | object | | Paramètres d'envoi |
| `workflow` | object | | Structure du workflow |

### Structure settings

```json
{
  "sendDays": ["monday", "tuesday", "wednesday", "thursday", "friday"],
  "sendHoursStart": 9,
  "sendHoursEnd": 18,
  "timezone": "Europe/Paris",
  "maxEmailsPerHour": 30
}
```

### Structure workflow

```json
{
  "nodes": [
    {
      "id": "node_1",
      "type": "start"
    },
    {
      "id": "node_2",
      "type": "email",
      "data": {
        "subject": "{{firstName}}, question rapide",
        "body": "Bonjour {{firstName}},\n\nJe me permets...",
        "trackOpens": true,
        "trackClicks": true
      }
    },
    {
      "id": "node_3",
      "type": "delay",
      "data": {
        "days": 3,
        "businessDays": true
      }
    },
    {
      "id": "node_4",
      "type": "email",
      "data": {
        "subject": "Suite à mon message",
        "body": "{{firstName}},\n\nJe reviens vers vous..."
      }
    },
    {
      "id": "node_5",
      "type": "end"
    }
  ],
  "edges": [
    {"from": "node_1", "to": "node_2"},
    {"from": "node_2", "to": "node_3"},
    {"from": "node_3", "to": "node_4"},
    {"from": "node_4", "to": "node_5"}
  ]
}
```

### Exemple complet

```bash
curl -X POST "https://api.fluenzr.co/v1/campaigns" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Prospection Q1 2025",
    "emailAccountId": "acc_xyz789",
    "settings": {
      "sendDays": ["monday", "tuesday", "wednesday", "thursday", "friday"],
      "sendHoursStart": 9,
      "sendHoursEnd": 18,
      "timezone": "Europe/Paris"
    },
    "workflow": {
      "nodes": [...],
      "edges": [...]
    }
  }'
```

## Modifier une campagne

```
PUT /campaigns/:id
```

> **⚠️ Attention**
Seules les campagnes en statut `draft` ou `paused` peuvent être modifiées.

### Exemple

```bash
curl -X PUT "https://api.fluenzr.co/v1/campaigns/cmp_abc123" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Prospection Q1 2025 - v2",
    "settings": {
      "maxEmailsPerHour": 50
    }
  }'
```

## Contrôler une campagne

### Démarrer

Lance une campagne en brouillon.

```
POST /campaigns/:id/start
```

```bash
curl -X POST "https://api.fluenzr.co/v1/campaigns/cmp_abc123/start" \
  -H "Authorization: Bearer flz_sk_xxx"
```

### Mettre en pause

Suspend une campagne active.

```
POST /campaigns/:id/pause
```

### Reprendre

Relance une campagne en pause.

```
POST /campaigns/:id/resume
```

### Archiver

Archive une campagne terminée.

```
POST /campaigns/:id/archive
```

## Gérer les contacts d'une campagne

### Ajouter des contacts

```
POST /campaigns/:id/contacts
```

```json
{
  "contactIds": ["con_abc", "con_def"],
  // OU
  "groupIds": ["grp_xyz"],
  // OU
  "filters": {
    "status": "active",
    "customFields": {
      "secteur": "Fintech"
    }
  }
}
```

### Exemple

```bash
curl -X POST "https://api.fluenzr.co/v1/campaigns/cmp_abc123/contacts" \
  -H "Authorization: Bearer flz_sk_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "groupIds": ["grp_xyz789"]
  }'
```

### Réponse

```json
{
  "success": true,
  "data": {
    "added": 150,
    "skipped": 5,
    "errors": []
  }
}
```

### Retirer des contacts

```
DELETE /campaigns/:id/contacts
```

```json
{
  "contactIds": ["con_abc", "con_def"]
}
```

### Lister les contacts

```
GET /campaigns/:id/contacts
```

| Paramètre | Description |
|-----------|-------------|
| `status` | Filtrer par statut (pending, in_progress, completed, replied, bounced) |
| `page` | Page |
| `perPage` | Éléments par page |
