# Campos personalizados

Os campos personalizados permitem adicionar informações específicas aos seus contatos além dos campos padrão.

## Por que usar campos personalizados?

### Personalização avançada

Use variáveis personalizadas em seus emails:
- `{{setor}}` → "Fintech"
- `{{tamanhoEmpresa}}` → "50-100 funcionários"
- `{{fonteContato}}` → "LinkedIn Sales Navigator"

### Segmentação

Filtre e segmente seus contatos de acordo com seus critérios de negócio.

### Relatórios

Analise seu desempenho de acordo com dimensões personalizadas.

## Criar um campo personalizado

### 1. Acessar configurações

1. Vá para **Configurações**
2. Clique em **Campos personalizados**
3. Clique em **+ Novo campo**

### 2. Configurar o campo

| Parâmetro | Descrição | Exemplo |
|-----------|-------------|---------|
| **Nome** | Identificador técnico (sem espaços) | `setor_industria` |
| **Label** | Nome de exibição | "Setor da Indústria" |
| **Tipo** | Tipo de dados | Texto, Número, Data, Lista |
| **Obrigatório** | Campo necessário? | Não |
| **Valor padrão** | Valor se não fornecido | - |

### 3. Tipos de campos disponíveis

| Tipo | Descrição | Caso de uso |
|------|-------------|----------|
| **Texto** | String de texto livre | Nota, comentário |
| **Número** | Valor numérico | Receita, contagem de funcionários |
| **Data** | Data (dia/mês/ano) | Data do primeiro contato |
| **Lista suspensa** | Escolha entre opções definidas | Setor, fonte |
| **Caixa de seleção** | Sim/Não | GDPR aceito |
| **URL** | Link web | Perfil do LinkedIn |

## Exemplos de campos úteis

| Campo | Tipo | Variável |

## Usar campos personalizados

### Em formulários

Os campos personalizados aparecem automaticamente em:
- O formulário de criação de contato
- O formulário de edição de contato
- Importação CSV (mapeamento de colunas)

### Em emails

Use a sintaxe `{{nome_do_campo}}` em seus templates:

```
Olá {{firstName}},

Como especialista em {{setor}}, você sabe que...

Com uma equipe de {{tamanhoEmpresa}} pessoas,
você certamente precisa...
```

### Em filtros

1. Vá para **Contatos**
2. Clique em **Filtros**
3. Selecione o campo personalizado
4. Defina a condição (igual a, contém, etc.)

## Importação com campos personalizados

### Formato CSV

Nomeie suas colunas com o nome técnico do campo:

```csv
email,firstName,lastName,setor_industria,tamanho_empresa
joao@example.com,João,Silva,Fintech,50-100
maria@example.com,Maria,Santos,SaaS,10-50
```

### Mapeamento

Durante a importação, o FluenzR detecta automaticamente as colunas correspondentes aos seus campos personalizados.

## Editar um campo personalizado

> **Aviso**
Editar um campo pode impactar dados existentes. Proceda com cautela.

### O que você pode editar

- Label (nome de exibição)
- Valor padrão
- Opções de lista suspensa (somente adicionar)

### O que você não pode editar

- Nome técnico (identificador)
- Tipo de campo

Para alterar o tipo, crie um novo campo e migre os dados.

## Excluir um campo personalizado

1. Vá para **Configurações** > **Campos personalizados**
2. Clique no ícone de exclusão
3. Confirme

> **Perigo**
A exclusão é irreversível. Todos os valores deste campo serão perdidos para todos os contatos.

## Melhores práticas

### Nomenclatura

| Recomendado | Evitar |
|-------------|-------|
| `setor_industria` | `Setor da Indústria` |
| `tamanho_empresa` | `size` |
| `data_primeiro_contato` | `data1` |

### Organização

- Limite o número de campos (10-15 no máximo)
- Prefira listas a campos de texto livre
- Documente o uso de cada campo
- Revise regularmente os campos não utilizados

### Valores padrão

- Use valores padrão relevantes
- "Não especificado" em vez de vazio para listas
- Facilite filtros com valores consistentes
