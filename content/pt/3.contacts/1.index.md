# Gestão de Contatos

Os contatos são o coração de suas campanhas. O FluenzR oferece ferramentas poderosas para importar, organizar e segmentar.

## Visão Geral

Um contato no FluenzR contém:

| Campo | Descrição | Obrigatório |
|-------|-------------|-------------|
| **Email** | Endereço de email único | Sim |
| **Nome** | Primeiro nome do contato | Não |
| **Sobrenome** | Nome de família | Não |
| **Empresa** | Nome da empresa | Não |
| **Cargo** | Posição/título | Não |
| **Telefone** | Número de telefone | Não |
| **Campos personalizados** | Dados adicionais | Não |

## Limites por plano

| Plano | Limite de contatos |
|------|-------------------|
| Free | 300 contatos |
| Starter | Ilimitado |
| Pro | Ilimitado |

## Funcionalidades

### Organização

- **Grupos**: Agrupe seus contatos por tema, origem ou critério
- **Tags**: Adicione etiquetas para facilitar a filtragem
- **Campos personalizados**: Crie seus próprios atributos

### Segmentação

- Filtre por grupo, tag ou valor de campo
- Combine múltiplos critérios
- Exporte segmentos

### Pipeline

Acompanhe o progresso de seus prospects com o pipeline CRM:
- Lead → Qualificado → Proposta → Negociação → Ganho/Perdido

### Importação/Exportação

- Importação CSV com mapeamento automático
- Exportação de todos os seus contatos ou uma seleção
- Detecção de duplicatas

## Status dos contatos

| Status | Descrição |
|--------|-------------|
| **Ativo** | Contato disponível para campanhas |
| **Descadastrado** | Clicou no link de descadastramento |
| **Bounced** | Email inválido (retorno de erro) |
| **Respondeu** | Respondeu a uma campanha |
| **Bloqueado** | Bloqueado manualmente |

## Melhores Práticas

### Qualidade dos dados

- Verifique emails antes da importação
- Use serviços de validação (Hunter, Neverbounce)
- Limpe regularmente os bounces

### Conformidade GDPR

- Colete consentimento quando necessário
- Respeite solicitações de descadastramento
- Documente suas fontes de coleta

### Organização

- Crie grupos lógicos desde o início
- Use convenções de nomenclatura claras
- Arquive contatos obsoletos
