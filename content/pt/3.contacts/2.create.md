# Criar um contato

Aprenda como adicionar contatos um por um no FluenzR.

## Criação manual

### 1. Acessar o formulário

1. Na barra lateral, clique em **Contatos**
2. Clique no botão **+ Adicionar um contato**

### 2. Preencher as informações

#### Informações obrigatórias

| Campo | Descrição |
|-------|-------------|
| **Email** | O endereço de email do contato (deve ser único) |

#### Informações opcionais

| Campo | Descrição | Variável |
|-------|-------------|----------|
| Nome | Primeiro nome do contato | `{{firstName}}` |
| Sobrenome | Nome de família | `{{lastName}}` |
| Empresa | Nome da empresa | `{{company}}` |
| Cargo | Posição/título | `{{jobTitle}}` |
| Telefone | Número de telefone | `{{phone}}` |
| Website | URL do site | `{{website}}` |

### 3. Adicionar a um grupo (opcional)

Selecione um ou mais grupos para organizar seu contato desde a criação.

### 4. Adicionar tags (opcional)

Adicione tags para facilitar a filtragem posteriormente.

### 5. Documentar consentimento (obrigatório)

O FluenzR exige que você documente a fonte de consentimento para cada contato. Este é um requisito legal (GDPR, CAN-SPAM):

| Fonte de Consentimento | Descrição |
|----------------|-------------|
| **Web Opt-in** | Contato registrado via formulário em seu site |
| **Evento** | Contato obtido em um evento (feira, conferência) |
| **Cartão de Visita** | Contato obtido através de troca de cartões de visita |
| **Rede Social** | Contato obtido via LinkedIn ou outra rede profissional |
| **Indicação** | Contato recomendado por terceiros |
| **Existente** | Contato já em sua base de dados (migração) |
| **Outro** | Outra fonte a especificar em notas |
| **Não Especificado** | Evite - pode causar problemas de conformidade |

Você também pode adicionar **notas de consentimento** para documentar as circunstâncias da coleta de forma mais precisa.

> **Aviso**
> Contatos sem fonte de consentimento documentada podem ser excluídos de certos envios por questões de conformidade.

### 6. Salvar

Clique em **Salvar** para criar o contato.

## Campos personalizados

Se você criou [campos personalizados](/docs/contacts/custom-fields), eles aparecem no formulário de criação.

Exemplo:
- Setor
- Tamanho da empresa
- Fonte do lead

## Validação de email

O FluenzR verifica automaticamente:

- **Formato**: O email deve ser válido (formato@dominio.extensao)
- **Unicidade**: O email não deve já existir em sua base de dados

> **Aviso**
Se o email já existir, você receberá uma mensagem de erro. Edite o contato existente ou use um email diferente.

## Melhores práticas

### Qualidade dos dados

- Preencha o máximo de campos possível para melhor personalização
- Verifique a ortografia do nome (inicial maiúscula)
- Use o formato padrão para empresas

### Organização

- Sempre atribua a um grupo relevante
- Use tags consistentes com sua estratégia
- Complete o pipeline se for um prospect qualificado

### GDPR & Conformidade

- **Sempre documente a fonte de consentimento** (obrigatório no FluenzR)
- A data de consentimento é registrada automaticamente
- Adicione notas se a fonte precisar de esclarecimento
- Entre em contato apenas com pessoas que deram seu consentimento
- Para conformidade CAN-SPAM (EUA), configure seu endereço físico nas configurações

## Após a criação

Uma vez criado o contato, você pode:

- **Adicioná-lo a uma campanha**: Da página do contato ou ao configurar a campanha
- **Ver seu histórico**: Emails enviados, aberturas, cliques, respostas
- **Editar suas informações**: Clique no contato para editar seu registro
- **Qualificá-lo**: Posicioná-lo no pipeline

## Criação rápida a partir de uma campanha

Você também pode criar contatos diretamente da tela de adição de contatos de uma campanha:

1. Abra uma campanha
2. Clique em **Adicionar contatos**
3. Clique em **Criar um novo contato**
4. Preencha o formulário rápido
5. O contato é adicionado automaticamente à campanha
